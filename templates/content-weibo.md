# 微博短文模板

## 格式规范

- **字数要求**：≤ 140 字（核心信息），可附加话题标签
- **结构**：话题标签 + 核心信息 + 行动呼吁 + 话题标签
- **风格**：简洁有力、口语化、适度使用emoji、善用数字和对比
- **话题标签**：前后各1-2个，格式 `#话题名#`
- **配图提示**：标注配图数量和类型建议

## 模板结构

```
#话题标签# {核心信息，简洁有力，一句话抓住注意力}。{补充说明或数据支撑，1-2句}。{行动呼吁} #话题标签#

[配图建议: {描述}，{数量}张]
```
