# 使用示例与输出格式

## 使用示例

### 示例一：生成月度报告

**用户输入**：
> 帮我分析一下2024年10月的宣传效果数据，生成月度分析报告。

**处理流程**：
1. 确认分析月份：2024年10月
2. 读取数据文件：
   - `_shared/assets/data/analytics/wechat-202410.csv`
   - `_shared/assets/data/analytics/weibo-202410.csv`
   - `_shared/assets/data/analytics/douyin-202410.csv`
   - `_shared/assets/data/analytics/summary-202410.csv`
3. 读取上月数据用于环比计算
4. 按报告模板生成完整月度报告

### 示例二：单平台分析

**用户输入**：
> 看看抖音上个月的传播数据表现怎么样？

**处理流程**：
1. 确认平台：抖音
2. 确认月份：上月
3. 读取抖音平台数据文件
4. 重点分析播放量、完播率、互动率等抖音核心指标
5. 输出抖音平台专项分析（含 Top 内容排名、趋势分析、优化建议）

### 示例三：主题分析

**用户输入**：
> 分析一下上个月各个主题方向的内容表现，哪个主题效果最好？

**处理流程**：
1. 读取上月全平台数据
2. 按内容主题分类汇总（交通法规、安全常识、警示案例、节日专题等）
3. 计算各主题的总阅读量、平均互动率等指标
4. 排名并输出主题效果对比分析

### 示例四：数据驱动策划

**用户输入**：
> 根据最近的传播数据，给下个月的宣传内容提一些建议。

**处理流程**：
1. 读取近1-2个月数据
2. 分析高效内容特征（主题、形式、发布时间）
3. 识别低效内容的共性问题
4. 结合趋势分析输出下月策划建议（主题方向、内容形式、发布节奏）

## 输出格式规范

### 月度报告格式

完整月度报告按 `assets/templates/report-monthly.md` 模板输出，包含以下章节：

1. **核心指标汇总** — 总体数据表 + 分平台数据表
2. **Top 内容排名** — 阅读量/播放量 Top 5 + 互动率 Top 5
3. **内容主题分析** — 各主题表现表 + 主题洞察
4. **趋势分析** — 周度趋势 + 发布时段效果
5. **优化建议** — 内容策略 + 发布策略 + 互动运营
6. **下月重点方向** — 具体可执行的计划建议

### 表格格式要求

数据表格统一使用 Markdown 格式：

```markdown
| 指标 | 本月 | 上月 | 环比变化 |
|------|------|------|---------|
| 总阅读量 | 125,600 | 108,300 | +16.0% |
| 总互动量 | 8,920 | 7,150 | +24.8% |
```

### 数值格式要求

- 整数：使用千位分隔符（如 125,600）
- 百分比：保留一位小数（如 16.0%）
- 环比变化：正数加"+"前缀（如 +16.0%），负数保留"-"（如 -5.3%）
- 大数值：超过 10,000 时可使用"万"为单位（如 12.5万）

### 建议格式要求

优化建议采用编号列表，每条建议包含：
1. **建议标题**：一句话概括
2. **数据依据**：支撑该建议的关键数据
3. **具体做法**：可操作的执行方案

示例：
> **1. 增加警示案例类视频内容**
> - 数据依据：本月警示案例类内容平均互动率 4.8%，高于整体均值 2.1%
> - 具体做法：下月计划发布 6-8 条警示案例短视频，优先选取本地真实案例
