# 指标定义

## 基础指标

| 指标 | 定义 | 适用平台 |
|------|------|---------|
| 阅读量 | 文章被打开阅读的总次数 | 微信 |
| 播放量 | 视频被播放的总次数 | 抖音 |
| 互动量 | 点赞+评论+转发/分享的总和 | 全平台 |
| 互动率 | 互动量/阅读量(播放量) × 100% | 全平台 |
| 完播率 | 完整看完视频的用户占比 | 抖音 |
| 新增关注 | 当月新增的粉丝/关注者数量 | 全平台 |

## 计算公式

- 平均互动率 = 总互动量 / 总阅读(播放)量 × 100%
- 环比变化 = (本月值 - 上月值) / 上月值 × 100%
- 篇均阅读量 = 总阅读量 / 发布篇数
