# 抖音短视频文案模板

## 格式规范

- **字数要求**：≤ 300 字
- **风格**：口语化、节奏感强、接地气、有"梗"或反转
- **结构**：hook（前3秒抓眼球）→ 核心内容 → 行动呼吁
- **语气**：像朋友聊天，避免公文腔，可适当用网络流行语
- **话题标签**：3-5个，放在文案末尾
- **特点**：开头必须有"钩子"，让用户停下来看

## 模板结构

```
{Hook：前3秒抓注意力的一句话，可用疑问/反转/冲突}

{核心内容：2-4句话讲清重点，口语化表达}

{行动呼吁：引导互动}

#话题1 #话题2 #话题3
```
