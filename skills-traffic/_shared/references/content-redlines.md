# 内容红线规则

> 适用于所有 Skill 生成和审核的交通安全宣传内容。

## 一、绝对禁止（触发即阻断）

### 1.1 政治敏感
- 不得出现对党和政府的负面评价
- 不得出现可能引发政治联想的隐喻或暗示
- 涉及国家领导人、国家政策时措辞必须准确
- 不得使用可能引发民族、宗教矛盾的表述

### 1.2 执法敏感
- 不得出现"暴力执法""以罚代管"等负面执法表述
- 不得泄露执法战术、布控信息、侦查手段
- 不得公开未审结案件的具体细节
- 不得出现有损交警部门正面形象的内容

### 1.3 暴力与血腥
- 事故现场描述不得过于血腥或暴力
- 不得使用恐吓性语言引发公众恐慌
- 不得发布可能引起心理不适的图文描述

### 1.4 隐私泄露
- 当事人姓名、车牌号、身份证号等必须脱敏处理
- 未成年人信息需特别保护
- 不得公开事故当事人的居住地址、工作单位等

### 1.5 受害者归责
- 不得出现责备受害者的表述或倾向
- 不得使用"自己不注意""活该"等归责性语言
- 体现"以人为本""生命至上"理念

## 二、需谨慎处理（触发警告）

### 2.1 绝对化用语
以下词汇应避免使用或需有充分依据：
- 最好、最佳、最先进、第一、唯一
- 100%、绝对、完全杜绝

### 2.2 数据引用
- "据统计""数据显示"后必须有可靠来源
- 不得编造或夸大统计数据
- 引用数据需标注年份和来源机构

### 2.3 法律法规引用
- 法条编号必须准确
- 罚款金额、记分分值必须与现行法规一致
- 法规名称使用全称或规范简称
- 注意法规是否有更新版本

### 2.4 歧视性表述
- 不得涉及性别歧视（如"女司机"等刻板印象）
- 不得涉及年龄歧视
- 不得涉及地域歧视
- 不得涉及职业歧视

## 三、内容导向要求

### 3.1 正面价值观
- 传递安全文明出行的正面价值观
- 体现交警部门服务公众的正面形象
- 鼓励守法出行，而非仅强调处罚

### 3.2 人文关怀
- 案例叙述中体现对生命的尊重
- 安全提示以"关心"而非"命令"的口吻
- 兼顾宣传效果与社会影响

### 3.3 版权合规
- 使用的图片、视频、音乐素材来源合法
- 引用他人内容需标注出处
- 不得涉及商业广告植入
