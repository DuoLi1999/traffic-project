# 风险分析方法论

> 本文档定义精准宣传方案助手的风险分析维度、指标和方法。

## 一、高风险区域分析

对 `highRiskAreas` 中的每个区域，从以下五个维度进行评估：

### 1.1 风险等级（riskLevel）

- **高**：年事故数 >= 50 或年致死人数 >= 5
- **中**：年事故数 30-49 或年致死人数 2-4
- **低**：年事故数 < 30 且年致死人数 < 2

### 1.2 事故数量（accidents）

- 统计该区域年度事故总数
- 计算占全市事故总量的百分比
- 与上年度对比分析趋势（如数据可用）

### 1.3 主要原因（mainCause）

- 提取该区域事故的主要违法/原因类型
- 分析原因与道路类型的关联性：
  - 高速公路：超速、疲劳驾驶为主
  - 城区路口：闯红灯、未按规定让行为主
  - 山区公路：超速、弯道越线为主
  - 农村道路：无证驾驶、酒驾为主
  - 商业区：非机动车违法为主

### 1.4 季节特征（seasonalPattern）

- 提取事故高发的时间模式
- 与 `monthlyTrend` 数据交叉验证
- 分类：
  - 季节性高发（冬季雨雾、冰雪天气）
  - 时段性高发（早晚高峰、午间/晚间）
  - 事件性高发（节假日、赶集日）

### 1.5 致死率

- 计算公式：`fatalities / accidents`
- 致死率 > 10% 为高致死区域，需重点关注
- 致死率排名用于确定宣传优先级

### 1.6 综合排名

综合以上维度，按以下权重排名：
- 致死率权重 40%
- 事故数量权重 30%
- 风险等级权重 20%
- 季节紧迫性权重 10%（当前季节匹配则加分）

## 二、高风险人群分析

对 `highRiskGroups` 中的每类人群，从以下维度进行评估：

### 2.1 涉事占比（percentage）

- 该群体涉事事故数占全市事故总量的百分比
- 占比 > 8% 为高涉事群体
- 占比 5%-8% 为中涉事群体

### 2.2 主要违法行为（mainViolation）

- 提取该群体最常见的违法行为
- 与 `byViolationType` 数据交叉分析
- 评估违法行为的致死风险等级

### 2.3 与区域的交叉分析

将人群与区域进行关联匹配：

| 人群 | 关联区域类型 |
|------|-------------|
| 外卖/快递骑手 | 城区商业区、城区路口 |
| 老年人 | 城区路口、农村道路 |
| 货车/重型车辆驾驶人 | 高速公路、山区公路 |
| 学生 | 学校周边、城区路口 |
| 农村地区驾驶人 | 农村道路、乡道 |

### 2.4 人群风险排名

按以下权重排名：
- 涉事占比权重 35%
- 涉事事故致死率权重 35%
- 违法行为危险性权重 20%
- 可干预性权重 10%（宣传可触达程度）

## 三、违法类型分析

### 3.1 致死率排名

对 `byViolationType` 中有致死数据的类型，计算每起违法的致死率：

```
致死率 = fatalities / count
```

按致死率从高到低排名，确定宣传优先级。通常排名：
1. 酒驾/醉驾（致死率最高）
2. 疲劳驾驶
3. 超速行驶

### 3.2 违法频次与致死的分离分析

- 高频低致死：如违法变道、刮擦 — 以提醒教育为主
- 低频高致死：如酒驾、疲劳驾驶 — 以警示震慑为主
- 高频高致死：如超速 — 最高优先级，多渠道密集宣传

## 四、时空风险图谱

### 4.1 时间分布分析

基于 `timeDistribution` 数据：

- 识别事故高发时段（事故数 top 3 时段）
- 识别高致死时段（致死率 top 3 时段）
- 注意：事故高发时段与高致死时段可能不一致
  - 高发时段（如17:00-19:00）：车流量大，轻微事故多
  - 高致死时段（如00:00-06:00）：事故少但致死率高（酒驾、疲劳）

### 4.2 天气关联分析

基于 `weatherRelated` 数据：

- 计算各天气条件下的事故占比
- 识别天气异常高风险（事故占比显著高于该天气出现频率）
- 雾天、冰雪天气通常是异常高风险天气

### 4.3 时空组合分析

将时间、天气、区域三维度交叉分析：
- 冬季 + 雾天 + 高速公路 = 追尾高风险
- 夏季 + 晚间 + 城区 = 酒驾高风险
- 节假日 + 农村道路 = 无证/酒驾高风险
- 工作日早晚高峰 + 商业区 = 非机动车违法高风险

用于精准确定宣传内容的投放时机和场景。
