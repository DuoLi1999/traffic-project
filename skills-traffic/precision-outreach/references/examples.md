# 用法示例

> 本文档提供精准宣传方案助手的典型使用场景和输出格式参考。

## 示例一：全量精准宣传分析

**用户输入：**
```
请基于2025年度事故数据，做一份全面的精准宣传分析报告。
```

**助手处理流程：**
1. 确认分析范围 — 全量分析，覆盖区域、人群、违法类型和时空维度
2. 读取 `_shared/assets/data/accident-data/2025-summary.json`
3. 对全部五个高风险区域和五类高风险人群逐一分析
4. 生成违法类型致死率排名和时空风险图谱
5. 为每个区域和每类人群分别生成定向宣传方案
6. 按报告模板输出完整报告

**预期输出结构：**
按 `assets/templates/outreach-report-template.md` 输出六个章节的完整报告。

## 示例二：单区域定向宣传方案

**用户输入：**
```
针对G15高速K100-K150段，帮我制定一个定向宣传方案。
```

**助手处理流程：**
1. 确认分析范围 — 单区域（G15高速K100-K150段）
2. 读取事故数据中该区域信息
3. 输出该区域的风险分析和定向宣传方案

**预期输出：**
```markdown
## G15高速K100-K150段 定向宣传方案

### 风险画像
- 风险等级：高
- 年度事故数：85起，致死8人，致死率9.4%
- 主要原因：超速、疲劳驾驶
- 季节特征：冬季雨雾天气高发

### 宣传方案
- **主题**：控速行驶，拒绝疲劳，安全到家
- **核心信息**：高速不超速，疲劳不上路
- **内容策略**：
  - 制作疲劳驾驶危害短视频（真实案例改编）
  - 服务区摆放休息提醒海报
  - 雨雾天气专题安全提示
- **投放渠道**：LED大屏、服务区海报、交通广播、高德/百度地图推送
- **投放时机**：冬季（11月-2月）密集投放，雨雾天气实时推送
- **配套活动**：联合高速交警开展"拒绝疲劳驾驶"主题宣传日
```

## 示例三：重点人群宣传方案

**用户输入：**
```
针对外卖骑手群体，制定精准宣传方案。
```

**助手处理流程：**
1. 确认分析范围 — 单人群（外卖/快递骑手）
2. 读取事故数据中该人群信息及关联区域
3. 输出该人群的风险分析和定向宣传方案

**预期输出：**
```markdown
## 外卖/快递骑手 定向宣传方案

### 风险画像
- 风险等级：高
- 涉事事故：180起，占全市事故总量的6.3%
- 主要违法行为：逆行、闯红灯、超速
- 关联高风险区域：商业步行街周边、城区路口

### 宣传方案
- **主题**：安全不迟到，生命比订单重要
- **内容风格**：同行共鸣型，口语化表达，骑手现身说法
- **内容策略**：
  - 拍摄骑手安全骑行纪录片
  - 制作"逆行的代价"系列短视频
  - 发布骑手安全骑行指南图文
- **投放渠道**：外卖平台App推送、骑手微信群、站点海报、抖音短视频
- **投放时机**：午间（11:00-13:00）和晚间（17:00-20:00）高峰前推送
- **配套活动**：
  - 联合美团、饿了么开展安全骑行培训
  - 设置安全骑行积分奖励机制
  - 站点张贴安全承诺书
```

## 输出格式说明

### 完整报告格式

完整分析报告应使用 `assets/templates/outreach-report-template.md` 模板，包含六个章节：

1. **数据概览**：年度数据总量和趋势
2. **高风险区域分析**：逐个区域分析 + 定向宣传方案
3. **高风险人群分析**：逐类人群分析 + 定向宣传方案
4. **关键违法类型分析**：致死率排名和分类策略
5. **时空风险分析**：时间分布 + 天气关联 + 组合分析
6. **综合宣传策略建议**：短期/中期/长期分层建议

### 格式要求

- 使用 Markdown 格式
- 数据使用表格呈现
- 百分比保留一位小数
- 大数字使用千分位分隔
- 所有数据标注来源（"据2025年度事故数据"）
- 方案建议使用可执行的清单形式
