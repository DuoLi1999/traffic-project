# 定向宣传策略指南

> 本文档定义区域定向和人群定向宣传方案的生成要素和策略规则。

## 一、区域定向宣传方案

针对每个高风险区域，方案应包含以下六个要素：

### 1.1 目标区域

- 明确标注区域名称和道路类型
- 标注风险等级（高/中/低）
- 简述区域特征（如城区路口、高速路段、农村道路等）

### 1.2 风险画像

- 年度事故数及致死率
- 主要事故类型和原因
- 高发时段和季节特征
- 涉及的主要人群

### 1.3 宣传主题

根据区域特征确定主题方向：

| 区域类型 | 建议主题方向 |
|---------|-------------|
| 高速公路 | 控速行驶、拒绝疲劳、恶劣天气应对 |
| 城区路口 | 遵守信号灯、礼让行人、安全过路口 |
| 山区公路 | 弯道减速、禁止越线、山路安全 |
| 农村道路 | 拒绝酒驾、持证上路、佩戴头盔 |
| 商业区 | 文明骑行、非机动车守法、关注行人 |

### 1.4 内容策略

- **核心信息**：一句话概括宣传核心（如"高速不超速，平安每一程"）
- **内容形式**：根据区域特点选择
  - 高速路段：LED大屏标语、服务区海报、广播提醒
  - 城区路口：短视频、朋友圈广告、路口电子屏
  - 农村道路：大喇叭广播、墙体标语、赶集日宣传
  - 商业区：外卖平台合作推送、骑手群公告
- **案例素材**：优先使用本区域的典型事故案例（需脱敏处理）

### 1.5 渠道选择

根据区域覆盖人群确定投放渠道：

| 优先级 | 渠道 | 适用区域 |
|--------|------|---------|
| 高 | 线下实地宣传（LED屏、海报、横幅） | 所有高风险区域 |
| 高 | 微信公众号/朋友圈 | 城区路口、商业区 |
| 中 | 抖音短视频 | 城区、商业区 |
| 中 | 交通广播 | 高速公路、国省道 |
| 中 | 村委广播/微信群 | 农村道路 |
| 低 | 微博 | 全域覆盖 |

### 1.6 投放时机

- 根据季节特征确定投放窗口（如冬季雨雾天气前1-2周启动）
- 根据时段特征确定推送时间（如早晚高峰前30分钟）
- 配合执法行动（如酒驾夜查前同步推送宣传）
- 节假日提前3-5天启动宣传

## 二、人群定向宣传方案

针对每类高风险人群，方案应包含以下六个要素：

### 2.1 目标人群

- 人群名称和特征描述
- 风险等级（高/中/低）
- 涉事事故数及占比

### 2.2 风险画像

- 主要违法行为
- 高发区域关联
- 事故特征（事故类型、伤亡情况）

### 2.3 宣传主题

根据人群特征确定主题方向：

| 目标人群 | 建议主题方向 |
|---------|-------------|
| 外卖/快递骑手 | 安全不迟到、生命比订单重要、文明骑行 |
| 老年人 | 过马路走斑马线、夜间出行穿反光衣 |
| 货车驾驶人 | 拒绝疲劳驾驶、定期检车、安全装载 |
| 学生 | 上下学安全、远离车辆盲区、安全过马路 |
| 农村驾驶人 | 拒绝酒驾、考证上路、一盔一带 |

### 2.4 内容风格

根据人群接受习惯调整风格：

| 目标人群 | 建议风格 | 语言特征 |
|---------|---------|---------|
| 外卖/快递骑手 | 同行共鸣 | 口语化、讲故事、同行现身说法 |
| 老年人 | 温馨关怀 | 简洁直白、大字体、子女视角 |
| 货车驾驶人 | 职业守护 | 朴实、讲利害、家人牵挂 |
| 学生 | 趣味互动 | 动画、游戏化、同学参与 |
| 农村驾驶人 | 乡土亲近 | 方言/接地气语言、身边事例 |

### 2.5 渠道选择

根据人群触媒习惯确定渠道：

| 目标人群 | 主要渠道 | 辅助渠道 |
|---------|---------|---------|
| 外卖/快递骑手 | 外卖平台App推送、骑手微信群 | 抖音短视频、站点海报 |
| 老年人 | 社区宣传栏、社区广播 | 子女朋友圈转发、电视 |
| 货车驾驶人 | 物流平台推送、驾驶人微信群 | 服务区海报、交通广播 |
| 学生 | 学校安全课、校园海报 | 家长微信群、抖音 |
| 农村驾驶人 | 村委广播、赶集日摆点 | 微信群、墙体标语 |

### 2.6 配套活动

宣传方案应搭配线下活动增强效果：

- **外卖骑手**：联合平台开展安全培训、设置安全骑行积分奖励
- **老年人**：社区安全讲座、"银发守护"一对一提醒
- **货车驾驶人**：物流企业安全会议、驾驶人安全承诺签名
- **学生**：校园安全体验课、交通安全知识竞赛
- **农村驾驶人**：集市宣传点现场答疑、摩托车免费检测活动

## 三、隐私保护规则

在制定和执行宣传方案时，必须遵守以下隐私保护要求：

1. **数据脱敏**：宣传材料中引用的事故案例必须脱敏，不得出现当事人姓名、车牌号、身份证号
2. **区域模糊**：区域分析精确到路段级别，不得涉及具体门牌号或小区名称
3. **人群泛化**：人群分析基于群体特征，不得指向特定个人
4. **数据安全**：事故原始数据仅用于分析，不得在宣传材料中直接展示原始数据表
5. **未成年保护**：涉及学生群体的宣传方案需特别注意未成年人隐私保护
6. **合规审查**：宣传方案发布前应经过内容红线审查（参照 `_shared/references/content-redlines.md`）
