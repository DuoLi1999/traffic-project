# 精准宣传方案助手

> 基于事故数据和违法数据，识别高风险区域与高风险人群，生成定向宣传方案。

## 功能概述

1. **高风险区域分析**：识别事故高发路段/区域，分析事故类型、原因及季节特征
2. **高风险人群分析**：识别事故涉及率高的群体，分析其主要违法行为及风险等级
3. **违法与时空分析**：按违法类型排序致死率，结合时间分布和天气关联绘制风险图谱
4. **定向宣传方案生成**：针对区域和人群分别生成宣传主题、内容策略、渠道选择和投放时机

## 触发条件

当用户消息中包含以下关键词或意图时激活本 Skill：

- 精准宣传、定向宣传
- 高风险区域、高风险人群、重点人群
- 事故分析、风险分析
- 要求基于事故数据制定宣传策略

## 数据来源

- 事故年度汇总：`_shared/assets/data/accident-data/2025-summary.json`
  - 包含字段：`overview`（总量与同比）、`monthlyTrend`（月度趋势）、`byAccidentType`（事故类型）、`byViolationType`（违法类型）、`highRiskAreas`（高风险区域）、`highRiskGroups`（高风险人群）、`weatherRelated`（天气关联）、`timeDistribution`（时间分布）

## 工作流程

### 第一步：确认分析范围

与用户确认：
- 分析维度：全量分析 / 区域分析 / 人群分析 / 专项分析
- 关注重点：特定区域、特定人群或特定违法类型
- 输出需求：完整报告 / 单项宣传方案 / 数据概览

### 第二步：读取事故数据

读取 `_shared/assets/data/accident-data/2025-summary.json`，提取以下关键字段：
- `overview`：年度事故总量、死亡人数、受伤人数、同比变化率
- `highRiskAreas`：高风险区域列表（含位置、类型、事故数、致死数、主要原因、季节特征）
- `highRiskGroups`：高风险人群列表（含群体名称、涉事数、占比、主要违法行为）
- `byViolationType`：违法类型分布（含致死数据）
- `timeDistribution`：各时段事故与致死分布
- `weatherRelated`：各天气条件下事故分布

### 第三步：风险分析

按照 `references/risk-analysis-guide.md` 中的方法论，从四个维度进行分析：
- 高风险区域分析（风险等级、事故数量、主要原因、季节特征、致死率）
- 高风险人群分析（涉事占比、主要违法行为、与区域的交叉分析）
- 违法类型分析（按致死率排名）
- 时空风险图谱（时间分布、天气关联）

### 第四步：生成定向宣传方案

按照 `references/outreach-strategy-guide.md` 中的策略指南，生成：
- **区域定向方案**：针对每个高风险区域的宣传主题、内容策略、渠道和时机
- **人群定向方案**：针对每类高风险人群的宣传主题、内容风格、渠道和配套活动

### 第五步：输出报告

使用 `assets/templates/outreach-report-template.md` 模板输出完整分析报告。
报告六个章节：
1. 数据概览 — 年度总量、同比趋势
2. 高风险区域分析 — 逐区域风险画像 + 定向方案
3. 高风险人群分析 — 逐人群风险画像 + 定向方案
4. 关键违法类型分析 — 致死率排名与分类策略
5. 时空风险分析 — 时间分布、天气关联、组合分析
6. 综合宣传策略建议 — 短期/中期/长期分层建议

## 约束规则

1. **数据驱动**：所有分析结论必须基于 `2025-summary.json` 中的数据，不得编造数据
2. **隐私保护**：分析报告中不得出现具体当事人信息，区域数据仅精确到路段级别
3. **内容安全**：宣传方案必须遵守 `_shared/references/content-redlines.md` 中的红线规则
4. **客观中立**：风险分析不得带有地域歧视、职业歧视或年龄歧视倾向
5. **正向引导**：宣传方案应以安全教育和正向引导为主，避免纯恐吓式宣传
6. **数据标注**：引用数据时必须标注来源和时间范围（如"据2025年度事故数据"）
7. **方案可行**：宣传建议应考虑基层执行能力，避免提出脱离实际的方案

## 输出格式

- 默认输出完整分析报告（Markdown 格式），结构参照报告模板
- 单项分析（如仅分析某区域或某人群）可输出对应章节
- 数据概览模式输出精简的 TOP 排名表格
- 宣传方案以可执行的清单形式呈现，包含负责人、时间节点等可操作信息
- 所有数据使用表格展示，百分比保留一位小数，大数字使用千分位分隔
