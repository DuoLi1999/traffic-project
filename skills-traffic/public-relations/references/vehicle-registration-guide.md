# 机动车业务知识指南

> 适用法规：《机动车登记规定》（公安部令第164号）、《道路交通安全法》及实施条例
> 生效日期：2022年5月1日

---

## 一、机动车年检（检验）

### 1.1 年检周期

根据车辆类型和使用年限，检验周期如下：

| 车辆类型 | 使用年限 | 检验周期 |
|---------|---------|---------|
| 9座及以下非营运小微型客车 | 6年以内 | 免于上线检验（第2、4年申领检验标志） |
| 9座及以下非营运小微型客车 | 6-10年 | 每2年检验1次 |
| 9座及以下非营运小微型客车 | 10-15年 | 每年检验1次 |
| 9座及以下非营运小微型客车 | 15年以上 | 每半年检验1次 |
| 营运载客汽车 | 5年以内 | 每年检验1次 |
| 营运载客汽车 | 5年以上 | 每半年检验1次 |
| 货车/大中型非营运客车 | 10年以内 | 每年检验1次 |
| 货车/大中型非营运客车 | 10年以上 | 每半年检验1次 |

### 1.2 六年免检政策

- **适用范围**：9座及以下非营运小微型客车（含面包车），且车辆未发生造成人员伤亡的交通事故。
- **2022年调整**：免检范围由6年内扩展至——第6年和第10年需上线检验，其余年份申领检验标志即可。
- **办理方式**：
  - 交管12123 APP → 机动车业务 → 免检车申领检验标志
  - 车管所窗口或交通管理服务站
- **所需材料**：
  1. 机动车行驶证
  2. 交强险保单（电子保单即可）
  3. 车船税完税证明（含在交强险中代缴的无需另行提供）

### 1.3 上线检验流程

1. 确认车辆无未处理违法记录（部分地区已取消此前置条件）
2. 前往具有资质的机动车检测机构
3. 外观检查 → 安全技术检验（制动、灯光、尾气等）→ 拍照
4. 检验合格后，核发检验合格标志
5. 检验不合格的，修复后可复检

### 1.4 逾期未检的后果

- 上路行驶将被处以 200 元罚款、记 3 分。
- 脱检期间发生事故，保险公司可拒赔交强险以外的部分。
- 连续 3 个检验周期未检验的，将被强制注销。

## 二、新车上牌（注册登记）

### 2.1 所需材料

1. 机动车所有人身份证明
2. 购车发票（机动车销售统一发票）
3. 车辆合格证（国产车）/ 进口凭证（进口车）
4. 交强险保单
5. 车辆购置税完税证明（或免税凭证）
6. 车船税完税证明

### 2.2 办理流程

1. **缴纳购置税**：税务局或电子税务局办理
2. **购买交强险**：保险公司办理
3. **前往车管所**：
   - 填写申请表
   - 拍摄车辆照片
   - 查验车辆（拓印车架号、核对发动机号等）
   - 选号（现场50选1 / 交管12123 APP自编或随机）
   - 缴费、领取行驶证和登记证书
   - 安装号牌

### 2.3 临时号牌

- **适用情形**：新车购买后尚未取得正式号牌期间。
- **类型**：
  - 行政辖区内临牌：有效期不超过 15 天
  - 跨辖区临牌：有效期不超过 30 天
- **可申领次数**：最多申领 3 次
- **注意**：临牌过期仍上路按"未悬挂号牌"处罚（罚款200元、记9分）。

## 三、机动车过户（转让登记）

### 3.1 所需材料

1. 现车主（买方）身份证明
2. 机动车登记证书
3. 机动车行驶证
4. 二手车销售统一发票或交易凭证
5. 车辆（需到场查验）

### 3.2 办理流程

1. 买卖双方到二手车交易市场开具交易发票
2. 前往车管所提交材料、查验车辆
3. 选号（可保留原号或重新选号）
4. 核发新行驶证，在登记证书上签注转让记录
5. 如跨辖区过户，需先在转出地办理转出手续

### 3.3 原号牌保留

- 原车主使用原号牌满 1 年以上，可申请保留原号牌。
- 保留时限：自办理转让之日起 2 年内可使用原号购置新车。
- 办理方式：在新车注册登记时申请使用保留号牌。

### 3.4 限制过户的情形

- 车辆达到强制报废标准的
- 车辆处于抵押、查封状态的
- 存在未处理的交通违法或交通事故的（部分地区已放开此限制）

## 四、机动车报废

### 4.1 强制报废标准

| 车辆类型 | 强制报废年限 |
|---------|-----------|
| 小型/微型非营运载客汽车 | 无强制报废年限（但行驶里程达60万公里引导报废） |
| 小型营运载客汽车 | 8年（可延长至10年） |
| 大型营运载客汽车 | 13年 |
| 出租客运汽车 | 8年 |
| 货运汽车（含半挂牵引车/全挂牵引车） | 15年 |
| 三轮汽车/低速货车 | 9年 |

### 4.2 报废流程

1. 将车辆交售给具有资质的报废机动车回收企业
2. 回收企业出具《报废机动车回收证明》
3. 持回收证明到车管所办理注销登记
4. 车管所核发注销证明

### 4.3 报废补贴

- 部分城市对提前报废老旧车辆（国三及以下排放标准）提供财政补贴。
- 补贴标准因地区而异，需咨询当地商务局或生态环境局。

## 五、其他常见业务

### 5.1 变更登记

以下情形需办理变更登记：

- 更换车身颜色
- 更换发动机
- 更换车身或车架
- 变更使用性质（如非营运转营运）
- 车主姓名或名称变更

办理时限：变更后 10 日内到车管所办理。

### 5.2 抵押/解除抵押登记

- 机动车作为贷款抵押的，需到车管所办理抵押登记。
- 贷款结清后，持解除抵押证明到车管所办理解除登记。
- 线上办理：部分地区支持通过交管12123办理解除抵押。

### 5.3 机动车电子凭证

- 检验标志电子凭证：通过交管12123 APP 申领，全国范围有效。
- 免粘贴纸质检验标志（2020年6月起全国推行）。
