# 交通事故处理知识指南

> 适用法规：《道路交通安全法》第70-76条、《道路交通事故处理程序规定》（公安部令第146号）
> 参考：各地交通事故快速处理办法

---

## 一、现场处置

### 1.1 事故发生后的法定义务

根据《道路交通安全法》第70条，发生交通事故后，当事人应当：

1. **立即停车**，保护现场
2. **抢救伤者**：如有人员受伤，立即拨打 120 急救电话并采取必要急救措施
3. **报警**：拨打 122 或 110 报警电话（造成人员伤亡或重大财产损失的，必须报警）
4. **设置警示标志**：
   - 城市道路：在车后 50 米以上放置警告标志
   - 高速公路：在车后 150 米以上放置警告标志
   - 夜间或雨雾天气应适当增大距离，并开启危险报警闪光灯
5. **保护现场**：不得移动车辆和相关物品（快速处理情形除外）

### 1.2 禁止行为

- **禁止逃逸**：造成交通事故后逃逸的，承担事故全部责任；构成犯罪的，处3年以上7年以下有期徒刑；因逃逸致人死亡的，处7年以上有期徒刑。
- **禁止故意破坏现场**：伪造现场、毁灭证据的，承担事故全部责任。
- **禁止饮酒后再报警**：事故发生后饮酒可能被认定为酒驾，影响责任判定。

### 1.3 现场记录

在确保安全的前提下，建议：

- 拍摄事故现场全景照片（含车辆位置、标志标线、信号灯）
- 拍摄车辆损坏部位细节照片
- 拍摄对方车辆号牌、驾驶证、行驶证
- 记录对方联系方式和保险信息
- 如有目击者，记录联系方式

## 二、快速处理（轻微事故自行协商）

### 2.1 适用条件

同时满足以下条件的，可以自行协商处理：

1. 仅造成车辆及财产损失（无人员伤亡）
2. 车辆可以移动
3. 当事人对事故事实及成因无争议
4. 各方均有有效的交强险

### 2.2 不适用快速处理的情形

- 造成人员伤亡的
- 当事人对事故事实有争议的
- 涉及酒驾、毒驾、无证驾驶等违法行为的
- 一方逃逸后返回的
- 碰撞建筑物、公共设施或其他设施需要赔偿的
- 车辆无号牌、无检验合格标志、无交强险的

### 2.3 快速处理流程

1. **拍照取证**：从前、后、左、右四个方向拍摄事故现场照片，照片应包含两车位置关系、碰撞部位、道路标志标线等
2. **撤离现场**：将车辆移至不妨碍交通的位置
3. **协商责任**：双方协商确定事故责任
4. **填写协议书**：
   - 使用交管12123 APP → 事故快处（推荐）
   - 或填写纸质《交通事故自行协商协议书》（车上应常备）
5. **办理理赔**：持协议书到保险公司办理理赔

### 2.4 交管12123 事故快处

- 打开 APP → 事故快处 → 拍照上传 → 在线确认责任 → 生成电子协议
- 全程线上操作，保险公司可直接调取电子协议
- 操作时限：事故发生后 24 小时内

## 三、需要报警的情形

### 3.1 必须报警的情形

- 造成人员伤亡的
- 当事人对事故事实有争议的
- 一方当事人离开现场的
- 涉及危险品运输车辆的
- 单方事故造成公共设施损坏的

### 3.2 报警方式

| 方式 | 号码/渠道 | 说明 |
|-----|---------|------|
| 电话报警 | 122（交通事故报警） | 全国统一交通事故报警电话 |
| 电话报警 | 110（公安报警） | 可转接122，24小时可用 |
| 电话报警 | 12123（交通安全服务） | 非紧急情况咨询 |
| 线上报警 | 交管12123 APP | 部分地区支持在线报警 |
| 高速报警 | 12122 | 高速公路事故报警及救援 |

### 3.3 报警后的流程

1. 交警到达现场 → 组织抢救伤者、勘查现场、收集证据
2. 当事人配合调查，如实陈述事故经过
3. 交警出具《道路交通事故认定书》（一般事故 10 日内，需检验鉴定的可延长）
4. 当事人根据事故认定书办理理赔或进入调解/诉讼程序

## 四、事故责任认定

### 4.1 责任类型

| 责任类型 | 说明 | 赔偿比例（参考） |
|---------|------|----------------|
| 全部责任 | 一方当事人的过错导致事故 | 全责方承担100% |
| 主要责任 | 一方过错为事故主要原因 | 主责方承担70%-80% |
| 同等责任 | 双方过错程度相当 | 各承担50% |
| 次要责任 | 一方过错为事故次要原因 | 次责方承担20%-30% |
| 无责任 | 一方无过错 | 无责方不承担赔偿（交强险内仍需赔付无责赔偿限额） |

### 4.2 对事故认定不服

- 可在收到事故认定书之日起 3 日内向上一级公安交管部门申请复核。
- 复核期限：受理后 30 日内作出复核结论。
- 复核期间任何一方提起诉讼的，终止复核。
- 事故认定书在民事诉讼中作为证据使用，法院可以根据其他证据作出不同认定。

## 五、理赔流程

### 5.1 保险报案

- 事故发生后应及时拨打保险公司客服电话报案（一般要求 48 小时内）
- 提供信息：保单号、事故时间/地点、事故简要经过、损失情况

### 5.2 交强险赔偿限额（2020年9月19日起）

| 赔偿项目 | 有责赔偿限额 | 无责赔偿限额 |
|---------|-----------|-----------|
| 死亡伤残 | 180,000 元 | 18,000 元 |
| 医疗费用 | 18,000 元 | 1,800 元 |
| 财产损失 | 2,000 元 | 100 元 |

### 5.3 理赔流程

1. **报案**：拨打保险公司电话报案
2. **定损**：保险公司安排定损员查勘定损（可到4S店或修理厂）
3. **维修**：在定损后进行车辆维修
4. **提交材料**：
   - 理赔申请书
   - 事故认定书或快处协议书
   - 驾驶证、行驶证复印件
   - 维修发票及清单
   - 人伤案件另需：医疗票据、诊断证明、伤残鉴定（如有）等
5. **审核赔付**：保险公司审核后赔付

### 5.4 理赔时效

- 保险公司应在收到齐全材料后 10 日内作出核定。
- 对属于保险责任的，应在与被保险人达成赔偿协议后 10 日内赔付。
- 理赔诉讼时效：2 年（自知道或应当知道保险事故发生之日起计算）。

## 六、特殊事故处理

### 6.1 高速公路事故

- 立即开启危险报警闪光灯
- 车上人员迅速转移到右侧护栏外
- 在来车方向 150 米以上放置警告标志
- 拨打 12122 高速报警电话
- 切勿在高速公路上停留争论或查看损失

### 6.2 涉及行人/非机动车的事故

- 机动车与行人/非机动车发生事故，机动车一方适用无过错责任原则。
- 即使行人/非机动车有过错，机动车方一般仍需承担不低于 10% 的赔偿责任。
- 行人/非机动车故意碰撞的除外。

### 6.3 单方事故

- 仅涉及自身车辆损坏的，如撞击护栏、路灯等。
- 需拍照留证后报警，同时向保险公司报案。
- 损坏公共设施的，需承担修复费用。
