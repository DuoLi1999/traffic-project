# 出行服务知识指南

> 本指南汇总公众出行中常见的交通服务类咨询
> 涉及政策可能因地区不同而有差异，请以当地最新规定为准

---

## 一、路况查询

### 1.1 查询渠道

| 渠道 | 说明 | 适用场景 |
|-----|------|---------|
| 高德地图/百度地图 | 实时路况、拥堵预测、路线规划 | 日常出行 |
| 交管12123 APP | 交通管制、施工信息 | 了解管制路段 |
| 12122 高速公路服务热线 | 高速路况、封路信息、救援服务 | 高速出行 |
| 省级交通广播 | 实时路况播报、事故信息 | 驾车收听 |
| 各地交警官方微博/微信 | 辖区路况、临时管制通告 | 获取本地信息 |
| 高速公路管理部门网站 | 收费标准、服务区信息 | 长途出行规划 |

### 1.2 高速公路信息

- **封路查询**：各省高速公路管理部门网站、12122热线、高德/百度地图实时更新
- **服务区查询**：导航软件可显示沿途服务区位置、设施及服务内容
- **收费查询**：各省ETC客服、高速公路管理网站可查询具体收费标准

## 二、限行政策

### 2.1 常见限行类型

| 限行类型 | 说明 | 典型城市 |
|---------|------|---------|
| 尾号限行 | 按车牌尾号限制通行日期 | 北京、天津、成都、西安等 |
| 区域限行 | 外地车辆在特定区域/时段限行 | 北京、上海、广州、深圳等 |
| 排放限行 | 按排放标准限制进入低排放区 | 多地设有国三/国四限行区 |
| 货车限行 | 货运车辆限时限区域通行 | 各大中城市普遍实施 |
| 临时限行 | 因重大活动、重污染天气等临时管控 | 视情况实施 |

### 2.2 限行查询方式

- **导航软件**：高德/百度地图在规划路线时会自动提示限行信息
- **当地交警公告**：交警微信公众号、官方网站发布限行通知
- **12123热线**：可咨询具体限行规则
- **限行日历**：部分城市交管APP提供限行日历功能

### 2.3 违反限行的处罚

- 一般处以 100-200 元罚款，部分城市记 3 分
- 各城市处罚标准不同，部分城市每4小时算一次违法
- 外地车辆需提前了解目的地限行政策，避免误入限行区域

## 三、ETC 办理

### 3.1 ETC 简介

ETC（Electronic Toll Collection，电子不停车收费系统）是高速公路电子收费系统，车辆通过收费站时无需停车即可完成缴费。

### 3.2 办理渠道

| 渠道 | 说明 | 特点 |
|-----|------|------|
| 银行网点 | 各大银行均可办理 | 通常赠送OBU设备、绑定信用卡/借记卡 |
| 线上办理 | 银行APP、微信/支付宝小程序 | 设备邮寄到家，自行安装激活 |
| 高速公路服务点 | 收费站/服务区办理点 | 现场安装，即办即用 |
| ETC助手/粤通卡等 | 各省ETC发行机构 | 部分提供储值卡模式 |

### 3.3 办理所需材料

- 个人用户：身份证 + 行驶证 + 银行卡（信用卡或借记卡）
- 单位用户：营业执照 + 行驶证 + 经办人身份证 + 单位银行账户

### 3.4 ETC 优惠政策

- **通行费折扣**：全国ETC用户享受不低于 95 折的通行费优惠
- **部分银行额外优惠**：首年免费、通行费返现、加油优惠等
- **高速通行效率**：ETC车道通行时间约为人工车道的 1/3

### 3.5 ETC 常见问题

- **更换车辆**：需到银行或ETC服务网点办理变更手续，重新绑定新车信息
- **注销**：可到银行或ETC发行机构办理注销，拆除OBU设备
- **扣费异常**：拨打 12122 或 ETC客服热线查询，可申请退费
- **设备故障**：到ETC服务网点检测或更换设备

## 四、高速免费政策

### 4.1 免费通行时间

全国高速公路对7座及以下小型客车实行免费通行的法定节假日：

| 节假日 | 免费时段 | 说明 |
|-------|---------|------|
| 春节 | 除夕00:00至假期最后一天24:00 | 共7天 |
| 清明节 | 假期第一天00:00至最后一天24:00 | 共3天 |
| 劳动节 | 假期第一天00:00至最后一天24:00 | 共5天 |
| 国庆节 | 假期第一天00:00至最后一天24:00 | 共7天 |

**注意**：元旦、端午节、中秋节高速不免费。

### 4.2 免费规则

- **适用车型**：7座及以下小型客车（含允许在公路上行驶的摩托车）
- **计时方式**：以车辆驶离出口收费站的时间为准
  - 即：在免费时段进入、免费时段结束后驶出的，需全程缴费
  - 在收费时段进入、免费时段内驶出的，免费
- **ETC车辆**：同样适用免费政策，免费时段内ETC不扣费

### 4.3 出行建议

- 免费首日/末日通常为出行高峰，建议错峰出行
- 免费最后一天接近24:00时，可在就近收费站先驶出再重新进入，避免全程收费
- 提前通过导航软件了解实时路况，合理规划路线
- 长途出行前检查车辆状况（轮胎、刹车、油液等），确保行车安全

## 五、其他出行服务

### 5.1 机动车号牌摇号/竞价

部分城市对小客车实行总量调控：

| 城市 | 调控方式 | 查询渠道 |
|-----|---------|---------|
| 北京 | 摇号（新能源排队） | 北京市小客车指标管理信息系统 |
| 上海 | 竞价拍卖 | 上海国拍网 |
| 广州 | 摇号 + 竞价 | 广州市中小客车指标调控管理信息系统 |
| 深圳 | 摇号 + 竞价 | 深圳市小汽车增量调控管理信息系统 |
| 杭州 | 摇号 + 竞价 | 杭州市小客车总量调控管理信息系统 |
| 天津 | 摇号 + 竞价 | 天津市小客车调控管理信息系统 |
| 海南 | 摇号（新能源不限） | 海南省小客车保有量调控管理信息系统 |

### 5.2 新能源汽车政策

- **免购置税**：符合条件的新能源汽车免征车辆购置税（政策延续中，具体截止日期以财政部公告为准）
- **不限行**：多数城市新能源汽车不受尾号限行限制
- **专用号牌**：新能源汽车使用绿色专用号牌（小型车渐变绿色，大型车黄绿双拼）
- **充电设施**：各地持续推进公共充电桩建设，导航软件可查找附近充电站

### 5.3 国际驾照与出境自驾

- 中国未加入《日内瓦道路交通公约》和《维也纳道路交通公约》，不签发国际驾照（IDP）。
- 出境自驾需根据目的地国家/地区的规定办理：
  - 部分国家认可中国驾照 + 公证翻译件
  - 部分国家/地区需申请当地临时驾照
  - 建议出行前咨询目的地驻华使领馆或租车公司
