# 使用示例

## 示例一：暴雨预警应急宣传

### 用户输入

> 暴雨橙色预警，今天下午到夜间市区及周边高速将有暴雨，预计持续8小时，部分路段可能积水。请生成应急宣传内容。

### 预期输出

生成完整应急宣传内容包：
1. **微信公众号推文**（500-800字）：预警信息 → 安全要点（减速慢行、避开积水路段、检查雨刮器等）→ 绕行建议 → 应急联系电话
2. **微博短文**（≤140字）：`#暴雨预警# 今日午后至夜间暴雨来袭，部分路段可能积水……请减速慢行，注意安全！#交通安全#`
3. **抖音文案**（≤200字）：紧迫感开头 → 3条核心安全要点 → 呼吁转发提醒
4. **预警通报**：橙色预警标准格式通报
5. **绕行建议**：积水易发路段的替代路线

### 文件输出

```
output/content/
├── 应急-暴雨-微信-20260211.md
├── 应急-暴雨-微博-20260211.md
├── 应急-暴雨-抖音-20260211.md
├── 应急-暴雨-通报-20260211.md
└── 应急-暴雨-绕行-20260211.md
```

## 示例二：交通管制应急宣传

### 用户输入

> G30连霍高速XX段因交通事故实行双向交通管制，预计管制4小时，请发布应急宣传。

### 预期输出

生成完整应急宣传内容包：
1. **微信公众号推文**：事故信息（不涉及伤亡细节）→ 管制路段说明 → 绕行方案 → 预计恢复时间 → 应急联系
2. **微博短文**：`#交通管制# G30连霍高速XX段因事故双向管制，预计4小时……请提前规划绕行路线 #出行提醒#`
3. **抖音文案**：紧迫感Hook → 管制信息 → 绕行要点 → 呼吁转发
4. **预警通报**：含管制路段、管制措施、预计恢复时间
5. **绕行建议**：至少两条替代路线方案

### 文件输出

```
output/content/
├── 应急-交通管制-微信-20260211.md
├── 应急-交通管制-微博-20260211.md
├── 应急-交通管制-抖音-20260211.md
├── 应急-交通管制-通报-20260211.md
└── 应急-交通管制-绕行-20260211.md
```

## 示例三：冰雪天气应急宣传

### 用户输入

> 明天起连续三天有冰雪天气，黄色预警，全市多条国省道和高速可能结冰，需要应急宣传。

### 预期输出

生成完整应急宣传内容包：
1. **微信公众号推文**：冰雪预警信息 → 安全要点（降低车速、保持车距、避免急刹急转、安装防滑链等）→ 结冰易发路段 → 绕行建议 → 应急联系
2. **微博短文**：`#冰雪预警# 明起连续三天冰雪天气，多条道路可能结冰……出行务必减速慢行、保持车距 #冬季行车安全#`
3. **抖音文案**：紧迫感Hook → 冰雪行车3大要点 → 呼吁收藏转发
4. **预警通报**：黄色预警标准格式通报，含持续时间和影响范围
5. **绕行建议**：结冰高风险路段的替代路线，附出行时间建议

### 文件输出

```
output/content/
├── 应急-冰雪-微信-20260211.md
├── 应急-冰雪-微博-20260211.md
├── 应急-冰雪-抖音-20260211.md
├── 应急-冰雪-通报-20260211.md
└── 应急-冰雪-绕行-20260211.md
```
