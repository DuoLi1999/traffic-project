# 素材标签分类法

本文档定义 media-hub 技能为交通安全宣传素材标引时使用的标签分类维度与规则。

## 标签维度

### 1. 事故类型

描述事故的具体形态。

| 标签 | 说明 |
|------|------|
| 追尾 | 前后车碰撞 |
| 侧翻 | 车辆侧翻或翻覆 |
| 碾压 | 车辆碾压行人或物体 |
| 单车事故 | 仅涉及一辆车的事故 |
| 多车连撞 | 三辆及以上车辆连续碰撞 |
| 坠桥坠崖 | 车辆坠落桥梁或悬崖 |

### 2. 违法种类

涉及的交通违法行为。

| 标签 | 说明 |
|------|------|
| 酒驾 | 饮酒或醉酒后驾驶 |
| 超速 | 超过规定速度行驶 |
| 闯红灯 | 违反交通信号灯通行 |
| 逆行 | 逆向行驶 |
| 疲劳驾驶 | 连续驾驶超时 |
| 超载 | 超过核定载重或载客量 |
| 分心驾驶 | 驾驶时使用手机等 |

### 3. 道路类型

事故或场景发生的道路环境。

| 标签 | 说明 |
|------|------|
| 高速公路 | 高速公路或快速路 |
| 城区道路 | 城市市区道路 |
| 农村道路 | 乡村或县道 |
| 山区公路 | 山区弯道、坡道 |
| 隧道 | 隧道内场景 |
| 路口 | 交叉路口区域 |

### 4. 天气条件

发生时的天气环境。

| 标签 | 说明 |
|------|------|
| 晴天 | 晴朗天气 |
| 雨天 | 降雨天气 |
| 雾天 | 大雾或团雾 |
| 冰雪 | 冰冻或降雪路面 |
| 大风 | 强风天气 |

### 5. 时间特征

事故或场景的时间段特征。

| 标签 | 说明 |
|------|------|
| 夜间 | 夜间行车场景 |
| 早高峰 | 早间通勤高峰时段 |
| 晚高峰 | 晚间通勤高峰时段 |
| 节假日 | 节假日出行场景 |

### 6. 涉及对象

事故或场景中涉及的主要对象。

| 标签 | 说明 |
|------|------|
| 货车 | 大型货运车辆 |
| 电动车 | 电动自行车或摩托车 |
| 行人 | 步行者 |
| 学生 | 学龄儿童或青少年 |
| 老年人 | 老年群体 |
| 校车 | 学生校车 |
| 危化品车 | 危险化学品运输车辆 |

### 7. 宣传用途

素材的宣传教育定位。

| 标签 | 说明 |
|------|------|
| 安全教育 | 安全知识科普类 |
| 警示教育 | 事故案例警示类 |
| 正能量 | 正面典型宣传类 |
| 节日宣传 | 节假日专题宣传类 |
| 法规宣传 | 法律法规解读类 |

## 标引规则

1. **数量要求**：每条素材标引 3-8 个标签，不得少于 3 个、不得超过 8 个。
2. **词语格式**：使用中文 2-4 字词语，不使用英文或长句。
3. **内容依据**：标签必须基于素材的实际内容（标题、描述、文件名等），不得臆测或编造。
4. **多维覆盖**：尽量从不同维度（如事故类型 + 道路类型 + 涉及对象）组合标签，提高检索覆盖率。
5. **优先选用**：优先使用上表中已定义的标签词；若上表未覆盖，可新增但应保持同等格式。
6. **去重与归一**：避免语义重复的标签（如同时使用"酒驾"和"醉驾"，应统一为"酒驾"）。
