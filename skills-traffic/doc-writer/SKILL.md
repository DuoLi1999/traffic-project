# doc-writer

交通安全宣传工作材料撰写技能，生成符合公文规范的各类工作材料。

## 功能概述

本技能与 content-producer（社交媒体内容生产）定位不同，专注于**内部公文和工作材料**的撰写：

1. **工作总结** — 月度/季度/年度宣传教育工作总结
2. **汇报材料** — 向上级汇报的工作进展、成效材料
3. **情况通报** — 交通安全形势分析通报、专项行动通报
4. **工作方案** — 专项宣传活动方案、整治行动宣传配合方案
5. **经验交流** — 宣传教育工作经验做法、典型案例总结

## 触发条件

用户消息包含以下意图关键词时激活：
- "写材料""撰写材料""起草材料"
- "写总结""工作总结""月度总结""年度总结"
- "写汇报""汇报材料""工作汇报"
- "写通报""情况通报""形势分析"
- "写方案""工作方案""活动方案"

## 工作流程

### Step 1：理解用户意图

从用户输入中提取以下要素：

| 要素 | 说明 | 默认值 |
|------|------|--------|
| 材料类型 | 总结/汇报/通报/方案/经验交流 | 必填 |
| 时间范围 | 涉及的工作时段 | 当月 |
| 重点内容 | 需要突出的工作亮点或重点 | 无 |
| 参考材料 | 已有的数据、素材或往期材料 | 无 |
| 篇幅要求 | 字数或页数要求 | 根据类型自动判断 |

### Step 2：确定文体规范

根据材料类型选择对应的公文格式：

- **工作总结**：总—分—总结构。开头概述整体情况，中间分条列述主要工作和成效（用数据说话），结尾指出不足和下一步计划。1500-3000字。
- **汇报材料**：问题导向结构。背景情况→主要做法→取得成效→存在问题→下一步打算。2000-4000字。
- **情况通报**：数据驱动结构。当前形势→数据分析→突出问题→工作要求。1000-2000字。
- **工作方案**：目标任务结构。指导思想→工作目标→组织架构→主要任务→时间安排→保障措施。2000-3000字。
- **经验交流**：叙述+提炼结构。背景→做法→成效→启示。1500-2500字。

### Step 3：注入数据支撑

从平台数据中提取可用的量化指标：
- 传播数据（发布量、阅读量、互动率等）
- 事故数据（同比变化、高风险区域治理成效等）
- 活动数据（开展次数、覆盖人数等）

确保材料中有充分的数据支撑，避免空话套话。

### Step 4：生成并输出

按公文格式输出完整材料，包含：
- 标题（含发文单位、事由、文种）
- 正文（按对应结构组织）
- 必要的数据表格
- 落款（单位、日期占位）

## 约束规则

1. **公文规范**：遵循党政机关公文格式标准，用语规范、结构清晰
2. **数据真实**：引用的数据必须来自提供的真实数据，不得编造
3. **政治正确**：涉及政策表述必须准确，与当前政策导向一致
4. **实事求是**：成绩不夸大、问题不回避，体现客观务实作风
5. **层次分明**：使用"一、（一）、1."三级标题体系
6. **语言风格**：庄重简洁，避免口语化表达和网络用语
7. **标注来源**：数据引用标注"据统计""数据显示"等，法规引用标注具体条款
