# 公文格式规范

## 通用格式要求

- 标题：发文单位 + 事由 + 文种（如"关于XX月交通安全宣传教育工作的总结"）
- 三级标题体系：一、（一）、1.
- 落款：单位全称 + 日期
- 数据引用标注"据统计""数据显示"等

## 各类型结构

### 工作总结（summary）

```
标题：关于{时间范围}交通安全宣传教育工作的总结
正文结构：
  一、工作开展情况（分条列述，含数据）
  二、取得的主要成效（量化指标）
  三、存在的不足
  四、下一步工作计划
篇幅：1500-3000字
```

### 汇报材料（report）

```
标题：关于{时间范围}交通安全宣传教育工作情况的汇报
正文结构：
  一、基本情况
  二、主要做法
  三、取得成效
  四、存在问题
  五、下一步打算
篇幅：2000-4000字
```

### 情况通报（briefing）

```
标题：关于{时间范围}全市道路交通安全形势的通报
正文结构：
  一、总体形势（数据概览、同比变化）
  二、突出特点（分类分析）
  三、高风险区域（表格列示）
  四、工作要求
篇幅：1000-2000字
```

### 工作方案（plan）

```
标题：关于开展{重点内容}专项宣传活动的工作方案
正文结构：
  一、指导思想
  二、工作目标（量化指标）
  三、组织架构
  四、活动时间
  五、主要任务（线上宣传、线下活动、警示教育）
  六、时间安排（表格：筹备期/预热期/集中期/总结期）
  七、保障措施
篇幅：2000-3000字
```

### 经验交流（experience）

```
标题：{重点内容}工作经验交流材料
正文结构：
  一、背景与起因
  二、主要做法
  三、取得成效
  四、经验启示
篇幅：1500-2500字
```
