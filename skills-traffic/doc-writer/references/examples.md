# doc-writer 使用示例

## 示例 1：生成月度工作总结

用户输入：
> 帮我写一下2026年1月的宣传教育工作总结，重点突出新媒体平台运营和交通安全进校园活动

技能参数：
```json
{
  "docType": "summary",
  "timePeriod": "2026年1月",
  "keyPoints": "新媒体平台运营和交通安全进校园活动"
}
```

## 示例 2：生成工作汇报

用户输入：
> 写一份向市局汇报的材料，内容是上半年宣传教育工作情况

技能参数：
```json
{
  "docType": "report",
  "timePeriod": "2026年上半年",
  "keyPoints": "宣传教育工作"
}
```

## 示例 3：生成形势通报

用户输入：
> 写一份1月份的交通安全形势通报

技能参数：
```json
{
  "docType": "briefing",
  "timePeriod": "2026年1月"
}
```

## 示例 4：生成活动方案

用户输入：
> 起草一个春运期间交通安全宣传活动方案

技能参数：
```json
{
  "docType": "plan",
  "timePeriod": "2026年春运期间",
  "keyPoints": "春运交通安全宣传"
}
```
