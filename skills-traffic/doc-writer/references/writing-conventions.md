# 公文写作规范

## 语言风格

- 庄重简洁，避免口语化表达和网络用语
- 使用书面语，不用"的话""其实""然后"等口语词
- 动词优先：用"开展""推进""落实"而非"进行了开展"
- 避免空话套话，每句话都应有实质内容

## 常用表述

### 开头段

- "根据工作安排，现将……汇报如下"
- "在……的正确领导下，紧紧围绕……的工作目标"
- "现将有关工作总结如下"

### 过渡段

- "在此基础上""与此同时""针对上述问题"
- "下一步，我们将重点做好以下几方面工作"

### 结尾段

- "以上汇报不当之处，请各位领导批评指正"
- "特此通报，请各单位高度重视，认真抓好落实"

## 数据引用规范

- 绝对数据：使用"共""累计""达到"等词
- 比较数据：使用"同比增长/下降X%""环比提升X个百分点"
- 来源标注：使用"据统计""数据显示""根据……统计"
- 不得编造数据，无数据时用"XX"占位

## 禁忌

- 不使用绝对化用语（最好、最佳、第一、唯一）
- 不使用网络流行语和表情符号
- 不出现具体个人姓名（用"张某""李某"脱敏）
- 政策表述必须与当前政策导向一致
