# 内容审核检查清单

## 审核流程

### 初审（格式与敏感词检查）
重点检查基本格式规范和敏感词。

### 复审（内容质量检查）
重点检查事实准确性、逻辑连贯性、表达质量。

### 终审（政策合规检查）
重点检查政治导向、法律合规、社会影响。

---

## 检查项目

### 一、敏感词与禁用表述

#### 1.1 政治敏感
- [ ] 不得出现对党和政府负面评价的表述
- [ ] 不得出现可能引发政治联想的隐喻或暗示
- [ ] 涉及国家领导人、国家政策时措辞必须准确
- [ ] 不得使用可能引发民族、宗教矛盾的表述

#### 1.2 执法敏感
- [ ] 不得出现"暴力执法""以罚代管"等负面执法表述
- [ ] 不得泄露执法战术、布控信息
- [ ] 不得公开未审结案件的具体细节
- [ ] 事故现场描述不得过于血腥或暴力
- [ ] 不得公开当事人真实姓名、身份信息（需脱敏处理）

#### 1.3 常见禁用词汇清单
以下词汇在宣传内容中应避免使用或需谨慎使用：

**绝对化用语**：
- 最好、最佳、最先进、第一、唯一、100%、绝对

**不当比喻**：
- 战争相关比喻（"向交通违法宣战"可用，但不宜过度军事化）
- 歧视性表述（涉及性别、年龄、地域、职业歧视）

**未经证实的数据表述**：
- "据统计""数据显示"后必须有可靠来源
- 不得编造或夸大数据

### 二、事实准确性

#### 2.1 法律法规引用
- [ ] 引用的法条编号正确
- [ ] 罚款金额、记分分值等处罚标准准确
- [ ] 法规名称使用全称或规范简称
- [ ] 注意法规是否有更新版本

#### 2.2 数据与统计
- [ ] 引用的统计数据需标注来源
- [ ] 年份、地点等信息准确
- [ ] 百分比、倍数等数值计算正确
- [ ] 不得使用未经核实的"网传"数据

#### 2.3 常识性检查
- [ ] 交通标志、标线描述准确
- [ ] 车辆类型、驾照类型等专业术语使用正确
- [ ] 地理信息（路名、路段编号）准确

### 三、内容质量

#### 3.1 标题检查
- [ ] 标题不涉及标题党（夸大、恐吓、低俗）
- [ ] 标题与内容相符
- [ ] 标题长度适中

#### 3.2 逻辑连贯性
- [ ] 论点与论据匹配
- [ ] 前后观点不矛盾
- [ ] 因果关系表述正确

#### 3.3 受众适宜性
- [ ] 语气风格适合目标受众
- [ ] 不包含可能引起恐慌的过度渲染
- [ ] 案例描述不过于血腥暴力

#### 3.4 格式规范
- [ ] 符合目标平台的格式要求（字数、排版）
- [ ] 无错别字、标点错误
- [ ] 配图建议合理

### 四、政策合规

#### 4.1 导向检查
- [ ] 内容传递正面价值观
- [ ] 不存在责备受害者的倾向
- [ ] 体现"以人为本""生命至上"理念
- [ ] 体现交警部门正面形象

#### 4.2 隐私保护
- [ ] 当事人信息已脱敏（姓名、车牌号、身份证号等）
- [ ] 事故现场照片/视频不暴露个人隐私
- [ ] 未成年人信息特别保护

#### 4.3 版权合规
- [ ] 使用的图片/视频/音乐素材来源合法
- [ ] 引用他人内容已标注出处
- [ ] 不涉及商业广告植入

---

## 审核结果分级

### 通过（PASS）
内容符合所有审核要求，可以发布。

### 警告（WARNING）
存在需要注意的问题，建议修改后发布。
- 数据来源未标注
- 个别表述不够严谨
- 格式有小瑕疵

### 阻断（BLOCK）
存在必须修改的严重问题，不得发布。
- 包含敏感词或禁用表述
- 法律法规引用错误
- 存在政治风险
- 泄露当事人隐私信息
- 内容导向有问题
