# 使用示例与输出格式

## 使用示例

### 示例 1：审核单篇文案

**用户输入：**
```
帮我审核一下这篇微信推文：output/wechat/2026-02-春运安全.md
```

**技能行为：**
读取文件内容，依次执行初审、复审、终审，输出完整审核报告。

---

### 示例 2：快速检查敏感词

**用户输入：**
```
看看这段文案有没有问题：

"某某路段昨日发生重大交通事故，造成3死5伤。张某某（男，35岁）驾驶车牌号为京A12345的小轿车闯红灯，是这起事故的罪魁祸首。交警部门已对该路段进行最严格的执法管控。"
```

**技能行为：**
对内联文本执行三级审核，重点标注隐私泄露（真实姓名、车牌号）、绝对化用语（"最严格"）、受害者归责倾向（"罪魁祸首"）等问题。

---

### 示例 3：合规检查

**用户输入：**
```
这篇法规宣传稿合规吗？看看引用的法条对不对。
```

**技能行为：**
重点执行复审（事实准确性）和终审（政策合规），校验法条编号、罚款金额、记分分值是否与现行法规一致。

---

### 示例 4：批量审核

**用户输入：**
```
审核一下 output/douyin/ 目录下本周的所有脚本。
```

**技能行为：**
逐个读取目录下的文件，分别执行三级审核，最后汇总输出批量审核报告。

---

```markdown

- **审核文件**：{文件路径}

---
```
