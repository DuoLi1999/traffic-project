# 使用示例

> content-producer 典型使用场景与预期输出示例。

## 示例 1：生成多平台文案

**用户输入**：
```
帮我写一组关于"春运安全出行"的宣传文案，三个平台都要
```

**技能行为**：
1. 识别主题为"春运安全出行"，平台为全部（微信+微博+抖音），内容类型为文案
2. 读取三个平台的模板和写作风格指南
3. 默认采用"温馨提醒"语气
4. 分别生成：
   - 微信推文（~1000 字，含导语、3 个分段、结尾呼吁、编辑信息）
   - 微博短文（≤ 140 字核心信息 + 话题标签）
   - 抖音文案（≤ 300 字，Hook 开头 + 口语化表达 + 话题标签）
5. 保存到 `output/content/` 目录：
   - `春运安全-微信-20260211.md`
   - `春运安全-微博-20260211.md`
   - `春运安全-抖音-20260211.md`

## 示例 2：指定单一平台与风格

**用户输入**：
```
写一篇微信公众号推文，主题是酒驾警示，用严肃警示的语气
```

**技能行为**：
1. 识别主题为"酒驾警示"，平台为微信，语气为"严肃警示"
2. 读取微信模板和写作风格指南中"严肃警示"相关规范
3. 生成微信推文：
   - 标题有力（如"酒驾的代价，你真的承受不起"）
   - 正文引用法规条文与真实案例（脱敏处理）
   - 数据引用标注来源
   - 语气庄重严谨
4. 保存到 `output/content/酒驾警示-微信-20260211.md`

## 示例 3：海报方案

**用户输入**：
```
做一组"开学季校园交通安全"的海报方案
```

**技能行为**：
1. 识别主题为"开学季校园交通安全"，内容类型为海报方案
2. 读取海报 schema 模板和写作风格指南
3. 生成 2-3 个差异化方案，例如：
   - 方案一（温馨风）：卡通插画，暖色调，面向小学生家长
   - 方案二（警示风）：实景+数据，冷色调，面向中学生
   - 方案三（创意风）：校园场景漫画，强调互动感
4. 输出 JSON 格式，包含每个方案的主标题、副标题、核心文案、视觉风格、尺寸、配色、配图建议
5. 保存到 `output/posters/开学季校园安全-海报方案-20260211.json`

## 示例 4：短视频脚本

**用户输入**：
```
帮我写一个关于"电动车戴头盔"的30秒抖音短视频脚本
```

**技能行为**：
1. 识别主题为"电动车戴头盔"，内容类型为视频脚本，时长 30 秒
2. 读取抖音模板和视频脚本 schema
3. 生成 5-6 个分镜：
   - 镜 1（0-3s）：Hook — "骑电动车不戴头盔？看完这个你就不敢了"
   - 镜 2（3-10s）：数据展示 — 头部伤害占比动画
   - 镜 3（10-18s）：对比演示 — 有/无头盔的冲击测试
   - 镜 4（18-25s）：真实案例 — 头盔救命的正面案例
   - 镜 5（25-30s）：行动呼吁 — "出门戴头盔，平安每一天"+ 品牌落版
4. 输出 JSON 结构化数据 + Markdown 分镜表
5. 保存到 `output/scripts/电动车头盔-脚本-20260211.md`
