# 内容生产助手（Content Producer）

## 功能概述

本 Skill 是交通安全宣传教育的内容创作引擎，根据用户提供的主题、平台要求，生成高质量的宣传内容：

1. **多平台文案生成**：微信公众号推文、微博图文、抖音短视频脚本
2. **海报设计方案**：提供海报文案、视觉建议、素材推荐
3. **内容定制**：根据目标受众（驾驶人、学生、老年人等）调整内容风格
4. **素材联动**：自动匹配 media-hub 中的相关素材

## 触发条件（When to Use）

当用户的输入匹配以下意图时激活本 Skill：

- 要求生成交通安全宣传内容
- 提到"写文案""写推文""写微博""写脚本"
- 提到"生成一篇...""写一篇关于..."
- 提到"做海报""设计海报"
- 提到"宣传内容""宣传文案"

**不激活**的情况：用户要求制定计划（由 plan-manager 处理）、审核内容（由 content-reviewer 处理）。

## 工作流程（Instructions）

### 步骤 1：解析创作需求

从用户输入中提取以下要素：

| 要素 | 说明 | 示例 |
|------|------|------|
| **目标平台** | 发布渠道 | 微信公众号、微博、抖音 |
| **内容主题** | 宣传话题 | 酒驾危害、雨天行车 |
| **目标受众** | 面向人群 | 驾驶人、学生、老年人 |
| **内容形式** | 表现形式 | 推文、图文、短视频 |
| **特殊要求** | 用户附加要求 | 语气、长度、风格 |

**如果用户未明确某个要素，主动询问。**

### 步骤 2：加载平台规范

根据目标平台，读取对应的模板和规范：

- 微信公众号：`assets/wechat-template.md`
- 微博：`assets/weibo-template.md`
- 抖音：`assets/douyin-template.md`

同时参考文案规范：`reference/copywriting-guide.md`

### 步骤 3：搜索相关素材

调用 media-hub skill 或直接搜索本地素材库，查找与主题相关的素材：

- 在 `data/materials/index.json` 中搜索相关标签
- 返回素材列表供内容创作参考
- 推荐可以在内容中引用的素材

### 步骤 4：生成内容

根据平台特点和主题要求，生成符合规范的内容：

#### 微信公众号推文

**结构要求**（参考 `assets/wechat-template.md`）：
- 标题：吸引点击（10-25字）
- 导语：快速点题（50-100字）
- 正文：层次分明，图文并茂
- 结尾：行动号召 + 互动引导
- 推荐字数：800-1500字

#### 微博内容

**结构要求**（参考 `assets/weibo-template.md`）：
- 文案：简洁有力（140-300字）
- 话题标签：2-4个相关话题
- 配图建议：明确配图需求
- 发布时间建议

#### 抖音脚本

**结构要求**（参考 `assets/douyin-template.md`）：
- 视频时长：15-60秒
- 开场黄金3秒：抓人眼球
- 内容结构：问题-案例-解决方案
- 字幕文案：配合口播
- 音乐建议：BGM选择
- 拍摄场景建议

#### 海报文案

**结构要求**：
- 主标题：冲击力强（5-10字）
- 副标题：补充说明（10-20字）
- 正文：核心信息（30-50字）
- 行动号召：明确引导
- 视觉建议：配图、色彩、排版

### 步骤 5：内容优化

根据 `reference/copywriting-guide.md` 进行优化：

- **标题优化**：吸引力、紧迫感、好奇心
- **开头优化**：黄金3秒法则
- **结构优化**：逻辑清晰、层次分明
- **结尾优化**：明确行动号召
- **语言优化**：通俗易懂、口语化

### 步骤 6：输出结果

1. 在对话中展示生成的内容
2. 保存到 `output/content/{日期}-{主题}-{平台}.md`
3. 提供后续建议：
   - 推荐配图素材（从 media-hub）
   - 建议发布时间
   - 提醒提交 content-reviewer 审核

## 文案风格指南

### 语气语调

根据不同受众调整：

| 受众 | 语气 | 特点 |
|------|------|------|
| 普通驾驶人 | 提醒式、亲切 | "您""请""注意" |
| 学生群体 | 教育式、活泼 | 生动案例、互动性强 |
| 老年群体 | 关怀式、耐心 | 大字体、简单明了 |
| 职业司机 | 专业式、严谨 | 数据支撑、法规引用 |

### 禁用词汇

- 过于血腥、恐怖的描述
- 可能引起恐慌的夸张表述
- 涉及个人隐私的具体信息
- 未经核实的统计数据

### 推荐表达方式

- 用案例说话：真实案例更有说服力
- 数据支撑：权威数据增强可信度
- 情感共鸣：引发读者共鸣
- 行动引导：明确告诉读者该怎么做

## 约束规则

### 内容生成规则
- 每篇内容必须有明确的标题
- 开头必须在3秒内抓住读者注意力
- 内容必须包含至少1个真实案例或数据
- 结尾必须有明确的行动号召
- 严格遵守平台字数限制

### 素材引用规则
- 引用素材时必须标注来源
- 不虚构事故案例（可使用通用场景描述）
- 图片/视频素材必须与实际内容匹配
- 推荐使用本地素材库中的素材

### 质量检查清单

生成内容后自查：
- [ ] 标题是否吸引人？
- [ ] 开头是否足够抓人？
- [ ] 内容逻辑是否清晰？
- [ ] 是否有具体案例或数据？
- [ ] 结尾是否有行动号召？
- [ ] 语言是否符合目标受众？
- [ ] 是否包含敏感信息？

## 使用示例

### 示例 1：生成微信公众号推文

```
用户: 帮我写一篇关于拒绝酒驾的微信推文

Agent 执行流程:
1. 确定平台: 微信公众号
2. 主题: 拒绝酒驾
3. 读取 wechat-template.md 和 copywriting-guide.md
4. 搜索 media-hub: "酒驾""醉驾"相关素材
5. 生成推文（含标题、导语、正文、结尾）
6. 推荐配图素材
7. 保存到 output/content/
```

### 示例 2：生成抖音脚本

```
用户: 写一个雨天行车的抖音短视频脚本

Agent 执行流程:
1. 确定平台: 抖音
2. 主题: 雨天行车安全
3. 读取 douyin-template.md
4. 生成脚本（含时长、分镜、台词、字幕、BGM建议）
5. 建议拍摄场景和素材
6. 输出完整脚本
```

### 示例 3：带特殊要求的内容

```
用户: 写一篇针对老年人的交通安全推文，语气要温和

Agent 执行流程:
1. 确定平台: 微信
2. 主题: 老年人交通安全
3. 受众: 老年群体
4. 特殊要求: 语气温和、字体清晰
5. 调整文案风格：使用"咱们""要注意"等亲切表达
6. 内容侧重：步行安全、过马路注意事项
7. 生成适合老年人的推文
```

### 示例 4：多平台内容生成

```
用户: 关于春运安全的宣传，给我生成微博和抖音两个版本

Agent 执行流程:
1. 主题: 春运安全
2. 平台1: 微博 -> 生成140字文案 + 话题标签
3. 平台2: 抖音 -> 生成15秒脚本
4. 确保两个版本核心信息一致
5. 分别输出两个平台的内容
```
