# 微博内容模板

## 微博结构

```
[文案正文（140-300字）]
[话题标签]
[配图建议]
[发布时间建议]
```

## 模板示例

### 文案正文

#### 格式一：短文案（140字以内）

```
[核心信息]！[数据/案例]。[提醒/倡议]。
#[话题1]# #[话题2]#
```

示例：
```
雨天行车，视线不清风险倍增！据统计，雨天事故率是晴天的1.5倍。减速慢行、保持车距，平安出行每一天！
#雨天行车安全# #交警提醒#
```

#### 格式二：长文案（140-300字）

```
[场景描述]
[案例/数据]
[原因分析]
[行动建议]
[互动提问]
#[话题1]# #[话题2]# #[话题3]#
```

示例：
```
【警惕！这个习惯可能致命】

很多司机都有这样的习惯：开车时看一眼手机。但就这一眼，相当于盲开几十米！

据统计，开车看手机发生事故的概率是普通驾驶的23倍。

生命只有一次，别让手机成了凶器。

开车时，请将手机静音或放入储物格。紧急电话请靠边停车后再接听。

你身边有开车看手机的人吗？转发提醒TA！

#开车别看手机# #交通安全# #交警正能量#
```

### 话题标签

选择原则：
- 相关性强
- 热度适中（避免完全没热度也避免过于热门被淹没）
- 2-4个为宜

类型：
- 交通安全类：#交通安全# #交警提醒# #平安出行#
- 主题类：#拒绝酒驾# #雨天行车# #疲劳驾驶#
- 活动类：#122全国交通安全日# #春运安全#
- 地域类：#[城市名]交警#

### 配图建议

| 数量 | 建议 |
|------|------|
| 1张 | 海报、长图、动图 |
| 2-4张 | 对比图、场景图、数据图 |
| 6张 | 九宫格（第9张为提示图） |
| 9张 | 完整九宫格 |

图片类型：
- 警示类：事故现场（适度打码）、违法抓拍
- 科普类：示意图、流程图
- 正能量：执勤照片、救助场景
- 海报：主题海报、提醒海报

### 发布时间建议

| 时段 | 特点 | 适合内容 |
|------|------|---------|
| 7:00-9:00 | 早高峰 | 出行提醒、路况 |
| 11:30-13:00 | 午休 | 案例警示 |
| 17:00-19:00 | 晚高峰 | 安全提醒 |
| 20:00-22:00 | 晚间 | 深度内容、互动 |
| 22:00-24:00 | 夜宵时段 | 拒绝酒驾 |

### 特殊格式

#### 提醒类微博

```
[emoji] 提醒 [emoji]

[具体提醒内容]

[行动建议]

[时间/地点说明]

#相关话题#
```

#### 案例类微博

```
[标题式开头]

[时间]：[地点]发生[事件]
[事件经过简述]
[后果/处理结果]
[警示提醒]

#相关话题#
```

#### 正能量微博

```
[温暖/感人场景描述]

[具体事迹]

[感人细节]

[点赞/呼吁]

#交警正能量# #暖心瞬间#
```

## 互动技巧

- 提问式结尾：增加评论互动
- 投票功能：设置简单二选一投票
- @相关账号：扩大传播
- 转发抽奖：提升转发量（需报备）

## 检查清单

发布前检查：
- [ ] 字数是否合适（不超500字）？
- [ ] 话题标签是否正确？
- [ ] 配图是否清晰？
- [ ] 有无错别字？
- [ ] 敏感信息是否处理？
- [ ] @对象是否正确？
