# 微信公众号推文模板

## 推文结构

```
[标题]
[封面图建议]
[导语]
[正文 - 多层次结构]
[结尾 - 行动号召]
[互动引导]
```

## 模板示例

### 标题（10-25字）

原则：吸引点击、制造悬念、引发共鸣

示例格式：
- 警示型：「X月X日，又一起悲剧发生...」
- 数字型：「90%的司机都不知道的X个细节」
- 疑问型：「你有没有犯过这个致命错误？」
- 情感型：「X岁司机的一句话，让人泪目」

### 导语（50-100字）

快速点明主题，引发阅读兴趣。

模板：
```
最近，[现象/事件]，引发了广泛关注。[一句话点题]。今天，我们就来聊聊[主题]。
```

### 正文结构

#### 方案一：案例驱动型

```
一、真实案例（200-300字）
   - 时间地点
   - 事件经过
   - 后果影响

二、原因分析（200-300字）
   - 直接原因
   - 深层原因
   - 常见误区

三、安全提示（300-500字）
   - 正确做法1
   - 正确做法2
   - 正确做法3

四、法规提醒（100-200字）
   - 相关法律条文
   - 处罚标准
```

#### 方案二：知识科普型

```
一、问题引入（100-200字）
   - 常见现象
   - 潜在危险

二、原理解析（300-400字）
   - 科学解释
   - 数据支撑
   - 专家观点

三、实用技巧（300-400字）
   - 技巧1 + 配图说明
   - 技巧2 + 配图说明
   - 技巧3 + 配图说明

四、常见误区（200-300字）
   - 误区1及纠正
   - 误区2及纠正
```

#### 方案三：情感共鸣型

```
一、故事引入（200-300字）
   - 人物介绍
   - 事件背景

二、情感渲染（300-400字）
   - 细节描写
   - 人物对话
   - 内心独白

三、主题升华（200-300字）
   - 普遍意义
   - 社会价值

四、行动倡议（100-200字）
   - 具体建议
   - 共同承诺
```

### 结尾（行动号召）

模板：
```
[总结核心信息]

[行动号召]
转发给身边的[目标人群]，让更多人[行动目标]！

[互动提问]
你/您的身边有[相关现象]吗？欢迎在评论区留言分享。
```

### 互动引导

- 点赞：「觉得有用请点赞」
- 在看：「点击在看让更多人看到」
- 转发：「转发给[人群]」
- 评论：「留言分享你的[经历/看法]」
- 关注：「关注我们获取更多安全知识」

## 排版建议

### 段落
- 每段3-5行，不超过手机一屏
- 段间距1行
- 重点内容加粗

### 配图
- 每500字配1张图
- 图片宽度900px
- 首图要吸引眼球
- 数据可视化用图表

### 字体
- 正文：16px
- 小标题：18px加粗
- 引用：14px灰色

## 字数建议

- 快速阅读：600-800字
- 深度内容：1200-1500字
- 专题报道：2000字以上

## 检查清单

发布前检查：
- [ ] 标题是否吸引人？
- [ ] 导语是否点题？
- [ ] 结构是否清晰？
- [ ] 配图是否匹配？
- [ ] 有无错别字？
- [ ] 行动号召是否明确？
