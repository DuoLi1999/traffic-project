# 大雾天气应急宣传模板

## 微信推文模板

```markdown
# 【紧急提醒】{颜色}色大雾预警！出行请注意安全

> ⚠️ {发布单位}发布{颜色}色大雾预警

## 🌫️ 预警详情

- **预警级别**：{颜色}色预警（{严重程度}）
- **发布时间**：{发布时间}
- **影响时间**：{开始时间} 至 {结束时间}
- **影响区域**：{详细区域列表}
- **预计能见度**：{能见度描述}

## ⚠️ 安全出行提示

### 🚗 驾车出行

1. **正确使用灯光**
   - 开启近光灯
   - 开启雾灯（前后雾灯）
   - 开启示廓灯
   - 开启前后位灯
   - ❌ 不要使用远光灯（会造成光幕）
   - ❌ 不要开双闪行驶（转向时无法示意）

2. **严格控制车速**
   - 能见度<200米：车速≤60km/h
   - 能见度<100米：车速≤40km/h
   - 能见度<50米：车速≤20km/h
   - 能见度<20米：就近驶离高速或进入服务区

3. **保持安全距离**
   - 与前车保持平时3倍以上的距离
   - 不要盲目超车
   - 不要跟车过近
   - 注意前车刹车灯

4. **谨慎驾驶操作**
   - 勤按喇叭，提醒其他车辆
   - 靠右行驶，以路边为参照
   - 不要在雾区停车（如必须停车，开启双闪并放置警示标志）
   - 听从交警指挥

5. **高速公路特别注意事项**
   - 进入雾区前减速
   - 不要紧急刹车
   - 不要随意变道
   - 从最近出口驶离或进入服务区
   - 红色预警时高速可能封闭，提前规划路线

### 🚶 行人出行

1. **穿戴醒目**
   - 穿颜色鲜艳的衣服
   - 最好穿戴反光衣物
   - 携带手电筒

2. **注意交通安全**
   - 能见度低，车辆难以及时发现行人
   - 过马路时格外小心
   - 不要在机动车道行走

3. **减少外出**
   - 大雾天气尽量减少外出
   - 必须外出时选择熟悉路线

## 🛣️ 路况信息

### 封闭/管制路段
{列出封闭或管制的路段}

### 能见度极低路段
{列出能见度极低需要特别注意的路段}

### 高速通行状态
{高速公路封闭/开放状态}

## 🆘 应急处置

### 雾中遇事故怎么办？
1. 立即开启双闪灯
2. 人员迅速撤离到护栏外
3. 在车后150米处放置警示标志
4. 拨打122报警

### 雾中迷路怎么办？
1. 不要慌张，减速慢行
2. 开启所有灯光
3. 以路边为参照
4. 寻找安全地点停车等待
5. 拨打122求助

## 📞 求助电话

- **报警求助**：110
- **交警指挥**：122
- **急救**：120
- **高速公路救援**：12122

## 📱 信息获取渠道

关注以下渠道获取最新路况：
- 官方微博：@{账号名}
- 微信公众号：{公众号名}
- 交通广播：{频率}
- 导航APP：查看最新路况
- 高速电子显示屏

---

**交警提醒：大雾天气，视线受阻，务必降速慢行！红色预警时请勿驾车出行！**

**转发提醒身边人！**
```

## 各级别预警内容差异

### 红色预警（能见度<50米）
- 强调"停止驾车出行"
- "已进入高速的请立即驶离或进服务区"
- 详细应急处置方法
- 语气最紧急

### 橙色预警（能见度<200米）
- 强调"减少不必要出行"
- 详细限速标准和驾驶技巧
- 重点路段提醒
- 语气紧急

### 黄色预警（能见度<500米）
- 强调"注意安全驾驶"
- 基本雾天驾驶技巧
- 路况提醒
- 语气警示
