# 冰雪天气应急宣传模板

## 微信推文模板

```markdown
# 【紧急提醒】{颜色}色冰雪预警！出行请注意安全

> ⚠️ {发布单位}发布{颜色}色{预警类型}预警

## ❄️ 预警详情

- **预警级别**：{颜色}色预警（{严重程度}）
- **发布时间**：{发布时间}
- **影响时间**：{开始时间} 至 {结束时间}
- **影响区域**：{详细区域列表}
- **预计情况**：{雨雪量描述}，气温降至{X}℃

## ⚠️ 安全出行提示

### 🚗 驾车出行

1. **做好车辆准备**
   - 检查防冻液、玻璃水
   - 检查轮胎气压和磨损情况
   - 准备防滑链（山区必备）
   - 准备铲雪工具、应急食物

2. **起步与行驶**
   - 慢起步，缓加油
   - 保持低速匀速行驶
   - 避免急加速、急刹车、急转向
   - 与前车保持2-3倍安全距离

3. **制动技巧**
   - 提前减速，轻点刹车
   - 手动挡车辆挂低档行驶
   - 自动挡车辆使用L档或雪地模式
   - 点刹代替急刹

4. **转向技巧**
   - 提前减速再转向
   - 慢打方向，避免急转
   - 转向不足时松油门
   - 转向过度时向滑动方向轻打方向

5. **坡道行驶**
   - 上坡前提前加速冲坡
   - 坡中不要换挡、不要停车
   - 下坡挂低档，利用发动机制动
   - 不要空挡滑行

6. **特别注意路段**
   - 桥梁、隧道口（易结冰）
   - 背阴路段（积雪不化）
   - 山区弯道（侧滑风险）
   - 上下坡路段

### 🚶 行人出行

1. **防滑措施**
   - 穿防滑鞋
   - 小步慢行
   - 双手不要插兜（保持平衡）

2. **过马路注意**
   - 车辆制动距离变长，提前过马路
   - 不要抢行
   - 注意路面结冰

## 🛣️ 路况信息

### 封闭/管制路段
{列出封闭或管制的路段}

### 易结冰路段
{列出容易结冰的路段}

### 绕行建议
{提供具体的绕行路线}

## 🆘 应急处置

### 车辆侧滑怎么办？
1. 松开油门和刹车
2. 向侧滑方向轻打方向盘
3. 车辆回正后再轻踩刹车
4. 不要猛打方向或急刹车

### 车辆被困怎么办？
1. 开启双闪灯
2. 在车后放置警示标志
3. 人员撤离到安全地带
4. 拨打122求助

## 📞 求助电话

- **报警求助**：110
- **交警指挥**：122
- **高速公路救援**：12122
- **道路救援**：{当地救援电话}

## 📱 信息获取渠道

关注以下渠道获取最新路况：
- 官方微博：@{账号名}
- 微信公众号：{公众号名}
- 交通广播：{频率}
- 导航APP：查看最新路况

---

**交警提醒：冰雪天气，安全第一！非必要不出行，出行务必谨慎驾驶！**

**转发提醒身边人！**
```

## 广播稿模板

```markdown
【交通广播应急稿】
时长：35-45秒

---

各位驾驶员朋友，现在插播一条紧急路况信息：

{发布单位}于{时间}发布{color}色{预警类型}预警，预计{时间范围}，{区域}将出现{天气描述}，气温降至{X}摄氏度。

交警紧急提醒：
冰雪路面行车，请务必做到：
第一，出发前检查车况，安装防滑链；
第二，起步慢、车速慢、转弯慢、刹车慢；
第三，与前车保持2到3倍安全距离；
第四，特别注意桥梁、隧道口等易结冰路段；
第五，如遇车辆侧滑，切记松油门松刹车，向侧滑方向轻打方向盘。

目前{路段}已实施交通管制，请过往车辆绕行{路线}。

最新路况请关注{频率}交通广播，或拨打122咨询。

安全提醒：冰雪天气，非必要不出行！

---
【播报注意】
- 语速适中，清晰有力
- 安全要点加重语气
- 保持冷静专业
```

## 各级别预警内容差异

### 红色预警（特别严重）
- 强调"停止非必要出行"
- 建议"寻找安全地点停车避险"
- 详细应急处置方法
- 语气最紧急

### 橙色预警（严重）
- 强调"尽量减少外出"
- 详细安全驾驶技巧
- 重点路段提醒
- 语气紧急

### 黄色预警（较重）
- 强调"注意安全驾驶"
- 基本冰雪驾驶技巧
- 路况提醒
- 语气警示

### 蓝色预警（一般）
- 提醒关注天气变化
- 基本防范建议
- 语气提醒
