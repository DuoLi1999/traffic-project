## 二、指标计算方法

### 互动率计算

```
微信互动率 = (点赞 + 在看 + 转发 + 收藏) / 阅读量 × 100%
微博互动率 = (转发 + 评论 + 点赞) / 阅读量 × 100%
抖音互动率 = (点赞 + 评论 + 分享) / 播放量 × 100%
```

### 传播指数计算

```python
传播指数 = 
  触达量得分 × 0.3 +
  互动量得分 × 0.3 +
  传播量得分 × 0.2 +
  转化量得分 × 0.2

其中：
得分 = min(100, 实际值 / 目标值 × 100)
```

### 增长率计算

```
环比增长率 = (本月 - 上月) / 上月 × 100%
同比增长率 = (今年同期 - 去年同期) / 去年同期 × 100%
```

- **采集内容**：阅读量、点赞、在看、转发、收藏

## 五、异常数据处理

### 识别异常值

```python
def detect_anomaly(values):
    Q1 = percentile(values, 25)
    Q3 = percentile(values, 75)
    IQR = Q3 - Q1
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR
    return [v for v in values if v < lower or v > upper]
```

### 异常类型

| 类型 | 表现 | 处理 |
|------|------|------|
| 数据缺失 | 某指标为0 | 标记并说明 |
| 数据异常 | 与其他数据矛盾 | 核实后修正 |
| 突发波动 | 短期大幅变化 | 标注原因 |
