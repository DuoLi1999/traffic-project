# 目标人群分析框架

## 一、人群分类维度

### 按风险等级分类

#### 极高风险人群
**特征描述**：
- 多次严重违法记录
- 造成过交通事故
- 存在明显危险驾驶行为

**细分类型**：
1. **习惯性违法者**
   - 违法频次：月均>3次
   - 违法类型：多种违法行为
   - 典型行为：超速、闯红灯、酒驾

2. **事故责任者**
   - 特征：过去2年内负有责任的事故
   - 风险：再次发生事故概率高
   - 需求：安全意识重塑

3. **高风险职业驾驶人**
   - 职业：货运、客运司机
   - 问题：疲劳驾驶、超载
   - 压力：时效压力、经济压力

#### 高风险人群
**特征描述**：
- 存在特定风险行为
- 特定时段或场景高风险

**细分类型**：
1. **年轻驾驶人群体**
   - 年龄：18-30岁
   - 驾龄：<3年
   - 高风险行为：超速、酒驾、分心驾驶
   - 触媒习惯：抖音、微博、B站
   - 心理特征：追求刺激、同伴影响大
   - 宣传策略：酷炫短视频、KOL影响、同伴证言

2. **老年交通参与者**
   - 年龄：60岁以上
   - 类型：老年驾驶人/老年行人
   - 问题：反应慢、规则意识弱
   - 触媒习惯：电视、微信、广播
   - 心理特征：固执、关注健康、重视家庭
   - 宣传策略：子女劝说、健康角度、简单直白

3. **外卖/快递骑手**
   - 职业：外卖、快递配送
   - 问题：赶时间、闯红灯、逆行
   - 压力：平台时效考核
   - 触媒习惯：骑手微信群、短视频
   - 宣传策略：平台合作、同行影响、案例警示

#### 中风险人群
**特征描述**：
- 偶发违法行为
- 特定场景风险

**细分类型**：
1. **新手驾驶人**
   - 驾龄：<1年
   - 问题：经验不足、应急能力差
   - 需求：驾驶技巧、应急处理

2. **女性驾驶人**
   - 特定问题：停车、复杂路况
   - 刻板印象：需要客观、尊重的宣传

3. **通勤驾驶人**
   - 特征：固定路线、固定时段
   - 问题：疲劳驾驶、路怒

### 按触媒习惯分类

#### 线上为主人群
- **年轻人**：抖音、小红书、B站
- **中年人**：微信、微博、今日头条
- **老年人**：微信、电视、广播

#### 线下为主人群
- 农村居民：村广播、宣传栏、集市
- 老年群体：社区活动、广场舞
- 职业司机：服务区、企业培训

### 按心理特征分类

#### 理性决策者
- 特征：注重数据、法规、逻辑
- 宣传策略：数据支撑、法规解读、案例分析

#### 情感驱动者
- 特征：受情感影响、关注家人
- 宣传策略：情感共鸣、家庭视角、正能量

#### 习惯依赖者
- 特征：按习惯行事、难改变
- 宣传策略：重复提醒、场景关联、渐进改变

## 三、重点人群详细画像

### 1. 年轻男性驾驶人（18-30岁）

#### 基本信息
- **占比**：驾驶人群的25%
- **违法占比**：严重违法人群的40%
- **高发违法**：超速、酒驾、飙车

#### 行为特征
- 夜间出行多
- 社交活动频繁
- 追求刺激感
- 同伴影响大

#### 心理特征
- 自信过度（过度自信驾驶技术）
- 侥幸心理（不会那么倒霉）
- 同伴压力（朋友都在做）
- 即时满足（追求当下快感）

#### 触媒习惯
- **主要渠道**：抖音、微博、B站、游戏
- **活跃时间**：晚上8点-凌晨2点
- **内容偏好**：刺激、酷炫、有趣
- **KOL影响**：大V、网红、游戏主播

#### 宣传策略建议
1. **内容形式**：
   - 15-30秒短视频
   - 酷炫特效+安全信息
   - 游戏化设计

2. **传播渠道**：
   - 抖音、快手
   - 游戏内广告
   - 电竞赛事合作

3. **关键信息**：
   - "酷的人懂得保护自己"
   - "真正的酷是平安回家"
   - 同龄人真实案例

4. **互动方式**：
   - 挑战赛
   - 转发抽奖
   - UGC内容征集

### 2. 职业货运司机（30-50岁）

#### 基本信息
- **群体规模**：XX万人
- **工作特点**：长时间驾驶、高疲劳度
- **高发违法**：疲劳驾驶、超载、超速

#### 行为特征
- 固定路线
- 时间压力大
- 独自工作多
- 经济压力大

#### 心理特征
- 职业认同强
- 家庭责任感强
- 注重实际利益
- 规则意识较强但受压力影响

#### 触媒习惯
- **主要渠道**：车载广播、微信群、服务区
- **活跃时间**：驾驶途中、服务区休息
- **内容偏好**：实用信息、行业资讯

#### 宣传策略建议
1. **内容形式**：
   - 广播节目
   - 微信长图文
   - 服务区海报

2. **传播渠道**：
   - 高速服务区
   - 物流企业合作
   - 货运平台

3. **关键信息**：
   - "你是家里的顶梁柱"
   - 疲劳驾驶成本计算
   - 同行案例分享

4. **激励方式**：
   - 安全驾驶奖励
   - 优秀司机评选
