# 交通违法处理流程指南

## 一、违法查询

### 查询方式

#### 1. 线上查询

**交管12123 APP（推荐）**
- 下载并注册"交管12123"APP
- 绑定驾驶证和车辆
- 首页点击"违法处理"
- 查看违法记录详情

**互联网交通安全综合服务管理平台**
- 访问 https://www.122.gov.cn
- 注册登录
- 进入违法处理业务
- 查询违法信息

**支付宝/微信城市服务**
- 支付宝：城市服务 → 车主服务 → 交通违法查询
- 微信：我 → 服务 → 城市服务 → 交通违法

#### 2. 线下查询
- 车管所自助查询机
- 交警大队违法处理窗口
- 部分银行网点自助终端

### 查询内容
- 违法时间、地点
- 违法行为
- 违法代码
- 罚款金额
- 记分分值
- 处理状态

## 二、罚款缴纳

### 缴纳时限
- **一般违法**：收到处罚决定书之日起15日内
- **现场处罚**：开具决定书后15日内
- **电子监控**：确认违法后15日内

⚠️ **逾期后果**：每日按罚款数额的3%加处罚款（滞纳金），最高不超过罚款本金。

### 缴纳方式

#### 线上缴纳（推荐）

**交管12123 APP**
1. 登录APP
2. 选择"违法处理"
3. 选择待处理违法记录
4. 确认违法信息
5. 在线支付罚款

**支持银行**
- 工商银行
- 建设银行
- 农业银行
- 中国银行
- 招商银行
- 等主流银行网银/手机银行

**第三方支付**
- 支付宝
- 微信支付

#### 线下缴纳

**银行网点**
- 持处罚决定书到指定银行
- 柜台或自助终端缴纳

**交警大队**
- 违法处理窗口直接缴纳

### 缴纳流程
1. 确认违法信息无误
2. 选择缴纳方式
3. 按提示完成支付
4. 保存缴款凭证
5. 确认处理状态更新

## 三、记分处理

### 记分规则
- 记分周期：12个月（从初次领证日期计算）
- 满分：12分
- 记分分值：12分、9分、6分、3分、1分

### 记分后果

| 记分情况 | 处理措施 |
|---------|---------|
| 1-11分 | 正常缴纳罚款，记分计入周期 |
| 达到12分 | 扣留驾驶证，参加满分学习 |
| 一个周期内2次达12分 | 满分学习+科目一、三考试 |
| 累计满24分 | 满分学习+科目一、三考试 |

### 满分学习

#### 学习方式
1. **现场学习**：到指定地点参加学习
2. **网络学习**：通过交管12123 APP在线学习
3. **自主学习**：自学后参加考试

#### 学习时长
- 记分12分：7天（每天不少于3小时）
- 记分24分及以上：14天

#### 考试恢复
- 学习完成后参加科目一考试
- 考试合格后清除记分，发还驾驶证
- 考试不合格继续学习考试

### 学法减分

#### 适用条件
- 驾驶证状态正常
- 当前记分周期内有记分
- 未被扣留、暂扣驾驶证

#### 减分方式
1. **网上学习**：3日内累计学满30分钟，考试合格减1分
2. **现场学习**：每次不少于1小时，考试合格减2分
3. **公益活动**：参加交通安全公益活动1小时减1分

#### 操作步骤
1. 登录"交管12123"APP
2. 选择"学法减分"
3. 选择学习方式
4. 完成学习并考试
5. 减分成功（一个周期最多减6分）

## 四、违法申诉

### 申诉情形
- 交通信号指示不一致
- 避让特种车辆（救护车、消防车等）
- 交通警察现场指挥与信号灯不一致
- 车辆被套牌
- 其他依法不予处罚的情形

### 申诉流程

#### 线上申诉
1. 登录"交管12123"APP
2. 进入违法记录详情
3. 点击"申诉"
4. 选择申诉原因
5. 上传相关证据
6. 提交申诉
7. 等待审核结果

#### 线下申诉
1. 准备材料：
   - 身份证原件及复印件
   - 驾驶证原件及复印件
   - 行驶证原件及复印件
   - 申诉理由说明
   - 相关证据材料
2. 前往违法处理窗口
3. 填写《交通违法行为申诉表》
4. 提交材料
5. 等待审核（一般15个工作日内）

### 申诉证据
- 行车记录仪视频
- 现场照片
- GPS轨迹
- 目击证人证言
- 其他相关证明

## 七、常见问题

### Q1: 可以异地处理违章吗？
**A:** 可以。全国已实现交通违法异地处理，可在违法行为发生地、机动车登记地或驾驶人户籍地处理。

### Q2: 别人的车违章扣我的分怎么办？
**A:** 可能是被套牌或车辆被借用。如被套牌，立即报警并申诉；如被借用，确认后处理。

### Q3: 不处理违章有什么后果？
**A:** 
- 产生滞纳金
- 车辆无法年检
- 驾驶证业务受限
- 影响个人征信（部分地区）

### Q4: 如何查询历史违法记录？
**A:** 通过交管12123 APP可查询最近3年的违法记录。

### Q5: 对处罚决定不服怎么办？
**A:** 可在收到处罚决定书60日内申请行政复议，或6个月内提起行政诉讼。
