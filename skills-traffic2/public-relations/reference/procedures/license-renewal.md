# 驾驶证换证流程指南

## 一、期满换证

### 办理条件
- 驾驶证有效期届满前90日内
- 驾驶证状态正常（未被扣留、吊销等）
- 无未处理的交通违法记录

### 所需材料
1. **身份证明**：身份证原件及复印件
2. **原驾驶证**：原机动车驾驶证
3. **体检证明**：县级或部队团级以上医疗机构出具的身体条件证明
4. **照片**：近期一寸白底免冠彩色照片（通常需3张）

### 办理流程

#### 方式一：线上办理（推荐）

**步骤：**
1. 下载"交管12123"APP
2. 注册并登录账号
3. 首页选择"驾驶证补换领"
4. 选择"期满换证"
5. 按提示上传材料照片
6. 选择取件方式（自取/邮寄）
7. 在线支付费用（工本费10元）
8. 等待审核和新证送达

**优点：**
- 无需排队
- 可以邮寄到家
- 7个工作日内办结

#### 方式二：线下办理

**步骤：**
1. 准备所需材料
2. 到指定医院进行体检
3. 前往车管所或交警大队业务大厅
4. 取号排队
5. 窗口提交材料
6. 缴费（工本费10元）
7. 等待制证
8. 领取新证

**办理地点：**
- 市级车管所
- 区县级交警大队
- 部分政务服务中心

### 办理时限
- **线上办理**：7个工作日内
- **线下办理**：当场办结或3个工作日内

### 费用标准
- 驾驶证工本费：10元
- 体检费：约20-50元（各医院不同）
- 快递费：到付（如选择邮寄）

### 注意事项

⚠️ **重要提醒：**
- 逾期未换证（1年内）：可正常换证，但不得驾驶机动车
- 逾期1-3年：需重新考科目一
- 逾期3年以上：驾驶证注销，需重新考取
- 换证前必须处理完所有交通违法

## 二、遗失补证

### 办理条件
- 驾驶证遗失或被盗
- 驾驶证损坏无法辨认

### 所需材料
1. 身份证原件及复印件
2. 近期一寸白底免冠彩色照片3张
3. 遗失声明（部分地方需要登报或现场填写）

### 办理流程

#### 线上办理
1. 登录"交管12123"APP
2. 选择"驾驶证补换领"
3. 选择"遗失补证"
4. 按提示操作
5. 支付工本费10元

#### 线下办理
1. 准备材料
2. 前往车管所
3. 填写《机动车驾驶证申请表》
4. 提交材料、缴费
5. 领取新证

### 注意事项
- 补证期间不得驾驶机动车（如原证已无法使用）
 -补领新证后，原证作废，如找回也不可使用
- 若怀疑被盗，建议报警

## 三、信息变更换证

### 适用情形
- 姓名变更
- 身份证号变更
- 住址变更
- 联系方式变更

### 所需材料
1. 身份证原件及复印件
2. 原驾驶证
3. 变更证明（如户口本变更页、派出所证明等）
4. 近期一寸白底免冠彩色照片3张

### 办理流程
1. 准备材料
2. 前往车管所
3. 提交变更证明材料
4. 填写申请表
5. 缴费、制证

## 四、降级换证

### 适用情形
- 自愿降低准驾车型
- 身体条件不符合原准驾车型要求

### 所需材料
1. 身份证原件及复印件
2. 原驾驶证
3. 体检证明（如因病降级）
4. 近期一寸白底免冠彩色照片3张

### 办理流程
1. 填写《机动车驾驶证申请表》
2. 说明降级原因
3. 提交材料、缴费
4. 领取新证

## 五、常见问题

### Q1: 可以异地换证吗？
**A:** 可以。全国已实现驾驶证异地换证，可在居住地车管所办理，无需返回发证地。

### Q2: 体检需要去哪里做？
**A:** 需到县级或部队团级以上医疗机构，或车管所指定的体检点。部分车管所内设体检点可一站式办理。

### Q3: 照片有什么要求？
**A:** 
- 近期6个月内拍摄
- 白底免冠
- 一寸（25mm×35mm）
- 不得着制式服装
- 面部无遮挡

### Q4: 换证可以代办吗？
**A:** 可以委托他人代办，需提供：
- 委托人身份证原件
- 代理人身份证原件及复印件
- 委托书
- 其他所需材料

### Q5: 线上办理多久能收到新证？
**A:** 一般3-7个工作日，具体视物流情况而定。

### Q6: 换证期间可以开车吗？
**A:** 如原证在有效期内可以驾驶，如已过期则不得驾驶，否则按无证驾驶处理。
