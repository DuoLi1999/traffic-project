# 交通事故处理流程指南

## 一、事故现场处置

### 1.1 立即停车

发生交通事故后，驾驶员应当：
- **立即停车**：开启双闪灯（危险报警闪光灯）
- **拉手刹**：防止车辆滑动
- **熄火**：关闭发动机
- **设置警示标志**：在车后适当位置放置三角警示牌

**警示标志放置距离**：
- 普通道路：车后50-100米
- 高速公路：车后150米以外
- 雨雾天气：适当延长距离

### 1.2 保护现场

- **标记位置**：用石头、粉笔等标记车辆位置（如需要移动车辆）
- **拍照取证**：拍摄事故现场全景、碰撞部位、刹车痕迹等
- **不要破坏现场**：不移动现场物品（除非救援需要）

### 1.3 抢救伤员

**如有人员受伤**：
1. **立即拨打120**：呼叫医疗急救
2. **初步救助**：在医护人员到达前进行必要救助
3. **不要随意移动伤者**：除非有二次伤害风险
4. **止血包扎**：如有出血进行简单止血

### 1.4 报警

**应当立即报警的情形**：
- 造成人员伤亡
- 较大财产损失（各地标准不同，一般5000元以上）
- 驾驶人无有效驾驶证
- 驾驶人有饮酒、吸毒嫌疑
- 车辆无号牌或假号牌
- 当事人不能自行移动车辆
- 一方当事人离开现场

**报警方式**：
- **电话报警**：拨打122（交警）或110
- **短信报警**：12110+区号后三位（部分地区支持）
- **APP报警**：交管12123 APP一键报警

**报警时需要说明**：
- 事故地点（详细位置）
- 人员伤亡情况
- 车辆受损情况
- 事故类型（追尾、碰撞等）
- 报警人姓名和联系方式

## 二、事故分类处理

### 2.1 轻微事故（无人员伤亡）

**适用条件**：
- 无人员伤亡
- 车辆可以自行移动
- 损失较小（各地标准一般为2000-5000元以下）
- 责任明确

**处理方式**：

#### 方式一：自行协商（私了）

1. **拍照取证**：
   - 两车位置关系
   - 碰撞部位
   - 路面标志标线
   - 对方车牌、驾驶证、行驶证

2. **交换信息**：
   - 姓名、联系方式
   - 车牌号
   - 保险公司、保单号

3. **撤离现场**：
   - 将车辆移至不妨碍交通的地方
   - 不要影响其他车辆通行

4. **协商赔偿**：
   - 协商一致后签订书面协议
   - 或各自向保险公司报案

⚠️ **注意事项**：
- 不要当场支付现金（避免后续纠纷）
- 对责任认定有争议不要私了
- 对方酒驾、无证不要私了
- 建议通过保险公司处理

#### 方式二：快速处理

使用"交管12123"APP快速处理：
1. 打开APP，选择"事故快处"
2. 拍摄现场照片（5张以上）
3. 上传照片，撤离现场
4. 协商或交警远程定责
5. 生成电子事故认定书

#### 方式三：交警处理

如无法协商一致，报警等待交警处理：
1. 保护现场等待交警
2. 交警到场勘查
3. 交警出具事故认定书
4. 按认定书处理

### 2.2 严重事故（有人员伤亡）

**处理流程**：
1. **立即停车**，开启双闪，设置警示标志
2. **抢救伤员**，拨打120
3. **立即报警**，拨打122或110
4. **保护现场**，等待交警和医护人员
5. **配合调查**，如实陈述事故经过

**注意事项**：
- 不要离开现场（可能构成肇事逃逸）
- 不要破坏现场
- 积极配合交警调查
- 及时通知保险公司

## 三、事故责任认定

### 3.1 认定依据

交警根据以下因素认定事故责任：
- 现场勘查情况
- 当事人陈述
- 证人证言
- 监控视频
- 车辆技术检验
- 酒精检测等

### 3.2 责任类型

| 责任类型 | 说明 | 赔偿比例（通常） |
|---------|------|----------------|
| 全部责任 | 一方完全过错 | 100% |
| 主要责任 | 一方过错较大 | 70% |
| 同等责任 | 双方过错相当 | 各50% |
| 次要责任 | 一方过错较小 | 30% |
| 无责任 | 无过错 | 0% |

### 3.3 常见事故责任划分

#### 追尾事故
- **一般情况**：后车全责
- **例外情况**：前车违法停车、倒车、溜车等

#### 变道事故
- **一般情况**：变道车全责
- **例外情况**：被变道车超速、故意撞车等

#### 路口事故
- 未按信号灯：违反信号灯方全责
- 未让行：未让行方全责
- 同时进入：右侧车辆优先

#### 逆行事故
- 逆行方全责

#### 倒车事故
- 倒车方全责

## 四、保险理赔流程

### 4.1 报案

**报案时限**：
- 一般事故：48小时内
- 人员伤亡：立即报案

**报案方式**：
- 保险公司客服电话
- 保险公司APP/微信公众号
- 保险代理人

**报案内容**：
- 保单号
- 事故时间、地点
- 事故经过
- 人员伤亡情况
- 车辆损失情况
- 对方车辆信息

### 4.2 定损

**定损方式**：
- 到保险公司定点修理厂定损
- 保险公司查勘员现场定损

**需要材料**：
- 事故认定书
- 驾驶证、行驶证
- 被保险人身份证
- 银行卡（理赔用）

### 4.3 理赔

**理赔流程**：
1. 提交理赔材料
2. 保险公司审核
3. 核定赔偿金额
4. 支付赔款

**理赔时效**：
- 小额案件：1-3个工作日
- 一般案件：5-10个工作日
- 复杂案件：不超过30日

## 五、法律责任

### 5.1 行政责任

| 违法行为 | 处罚 |
|---------|------|
| 肇事逃逸（不构成犯罪） | 罚款200-2000元，可并处15日以下拘留，吊销驾照 |
| 酒驾 | 暂扣6个月驾照，罚款1000-2000元 |
| 醉驾 | 吊销驾照，5年内不得重新取得，刑事责任 |
| 无证驾驶 | 罚款200-2000元，可并处15日以下拘留 |

### 5.2 刑事责任

**交通肇事罪**（刑法第133条）：
- 死亡1人或重伤3人以上，负主要或全部责任：3年以下有期徒刑或拘役
- 逃逸或有其他恶劣情节：3-7年有期徒刑
- 因逃逸致人死亡：7年以上有期徒刑

**危险驾驶罪**（刑法第133条之一）：
- 醉驾：拘役，并处罚金
- 追逐竞驶（飙车）：拘役，并处罚金
- 严重超载/超速：拘役，并处罚金

### 5.3 民事责任

- 赔偿对方车辆损失
- 赔偿对方人身损害（医疗费、误工费、护理费等）
- 赔偿对方财产损失

## 六、事故处理注意事项

### 6.1 现场注意事项

✅ **应该做**：
- 立即停车、保护现场
- 抢救伤员、及时报警
- 拍照取证、记录信息
- 配合调查、如实陈述

❌ **不应该做**：
- 不要逃逸（加重责任）
- 不要破坏现场
- 不要与对方争吵
- 不要轻易私了（复杂情况）
- 不要承认全责（等待认定）

### 6.2 证据保全

- **拍照**：事故现场、车辆位置、碰撞部位、路面痕迹
- **记录**：对方信息、证人信息
- **保存**：行车记录仪视频、监控录像
- **收集**：医疗票据、维修发票等

### 6.3 后续处理

- **维修车辆**：到正规修理厂维修
- **伤残鉴定**：如有伤残需做鉴定
- **调解赔偿**：可申请交警调解或法院诉讼
- **取回车辆**：事故处理完毕后取回被扣车辆

## 七、常见问题

### Q1: 事故后对方逃逸怎么办？
**A:** 立即报警，尽可能记住对方车牌、车型、颜色、逃逸方向，拍照记录现场。

### Q2: 对事故认定不服怎么办？
**A:** 可在收到认定书3日内向上一级交管部门申请复核。

### Q3: 对方不赔偿怎么办？
**A:** 可申请交警调解，或向法院提起民事诉讼，同时可申请财产保全。

### Q4: 事故后驾照被扣多久能取回？
**A:** 一般事故调查结束后可取回；如涉及检验鉴定，需等鉴定结论。

### Q5: 没有保险或保险过期怎么办？
**A:** 需自行承担赔偿责任；如交强险过期，需补缴并接受处罚。

### Q6: 事故中手机没电无法报警怎么办？
**A:** 向过往车辆求助借用电话，或到附近商店、住户求助。
