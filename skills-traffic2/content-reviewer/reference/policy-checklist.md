# 政策合规检查清单

## 一、宣传政策合规

### 1.1 导向正确
- [ ] 符合社会主义核心价值观
- [ ] 弘扬正能量
- [ ] 体现交警队伍良好形象
- [ ] 符合公共安全宣传基调

### 1.2 内容基调
- [ ] 警示教育为主，不渲染血腥
- [ ] 正面引导为主，不制造恐慌
- [ ] 教育意义明确，不流于形式
- [ ] 受众友好，不产生抵触情绪

### 1.3 主题适宜
- [ ] 符合当前宣传工作重点
- [ ] 不与敏感时段冲突
- [ ] 不涉及敏感话题
- [ ] 时机选择恰当

## 二、法律法规合规

### 2.1 法律条款引用
- [ ] 引用条款准确无误
- [ ] 使用最新有效条款
- [ ] 条款解释正确
- [ ] 出处标注清晰

#### 常用法律条款速查

**《道路交通安全法》**
- 第22条：饮酒、服用药物不得驾驶机动车
- [其他常用条款 - 占位]

**《刑法》**
- 第133条：交通肇事罪
- 第133条之一：危险驾驶罪
- [其他常用条款 - 占位]

**《道路交通安全法实施条例》**
- [常用条款 - 占位]

### 2.2 处罚标准
- [ ] 处罚标准准确
- [ ] 与最新规定一致
- [ ] 区分不同违法情形
- [ ] 注明法律依据

### 2.3 程序合规
- [ ] 执法程序描述正确
- [ ] 不误导公众认知
- [ ] 体现执法规范性
- [ ] 维护法律权威

## 三、保密与隐私

### 3.1 案件信息保密
- [ ] 不披露侦查细节
- [ ] 不公开未结案件详情
- [ ] 不泄露证人信息
- [ ] 不披露技术侦查手段

### 3.2 个人隐私保护
- [ ] 车牌号打码处理
- [ ] 人脸打码处理
- [ ] 姓名使用化名或省略
- [ ] 住址信息模糊化
- [ ] 身份证号、手机号等隐藏

### 3.3 涉密信息
- [ ] 不含涉密单位信息
- [ ] 不含内部工作部署
- [ ] 不含敏感行动计划
- [ ] 不含可能危害安全的信息

## 四、画面内容适度

### 4.1 血腥度控制
- [ ] 不过度展示血腥画面
- [ ] 尸体等画面模糊处理
- [ ] 避免特写惨烈细节
- [ ] 警示不等于恐怖展示

### 4.2 画面处理标准
- [ ] 血腥画面：模糊/马赛克/黑白
- [ ] 伤者面部：打码/侧影/遮挡
- [ ] 敏感部位：裁剪/模糊/遮挡
- [ ] 家属情绪：保护性处理

### 4.3 心理影响评估
- [ ] 不会引发过度恐慌
- [ ] 不会造成心理创伤
- [ ] 不会引起不适反应
- [ ] 适合目标受众观看

## 五、舆情风险防控

### 5.1 敏感话题规避
- [ ] 不涉及敏感事件
- [ ] 不触碰争议话题
- [ ] 不涉及民族宗教敏感内容
- [ ] 不涉及地域歧视

### 5.2 表述风险
- [ ] 无不当对比（如中外执法对比）
- [ ] 无可能引发对立的内容
- [ ] 无可能被断章取义的表述
- [ ] 无可能引发误解的表述

### 5.3 舆情预判
- [ ] 评估可能的社会反应
- [ ] 预判可能的质疑点
- [ ] 准备回应口径（如需要）
- [ ] 请示汇报（高风险内容）

## 六、平台规范合规

### 6.1 微信公众号
- [ ] 符合微信公众平台规范
- [ ] 不含诱导分享内容（如"不转不是中国人"）
- [ ] 不含虚假夸大宣传
- [ ] 符合内容安全标准

### 6.2 微博
- [ ] 符合微博社区公约
- [ ] 话题标签使用规范
- [ ] 不涉及微博敏感词
- [ ] 符合字数限制

### 6.3 抖音
- [ ] 符合抖音社区规范
- [ ] 画面内容合规
- [ ] 音乐版权合规
- [ ] 不含违规营销

## 七、特殊内容审核

### 7.1 事故报道
- [ ] 事实准确（不夸大、不缩小）
- [ ] 时间地点准确
- [ ] 伤亡数据准确
- [ ] 原因分析客观
- [ ] 责任认定依据权威通报

### 7.2 执法报道
- [ ] 执法主体明确
- [ ] 执法过程合规
- [ ] 不暴露执法策略
- [ ] 维护执法权威

### 7.3 正能量宣传
- [ ] 事迹真实
- [ ] 人物真实
- [ ] 不夸大、不虚构
- [ ] 情感真挚自然

## 八、审核流程

### 常规内容
1. AI初审 → 2. 人工复审 → 3. 负责人终审 → 4. 发布

### 敏感内容
1. AI初审 → 2. 人工复审 → 3. 部门审核 → 4. 领导审批 → 5. 发布

### 特别敏感内容
1. 请示报告 → 2. 上级审批 → 3. 按批复执行

## 九、问题定级标准

| 问题类型 | 定级 | 处理要求 |
|---------|------|---------|
| 政治敏感 | 致命 | 禁止发布 |
| 泄露秘密 | 致命 | 禁止发布 |
| 严重失实 | 致命 | 禁止发布 |
| 隐私泄露 | 严重 | 必须修改 |
| 画面过度 | 严重 | 必须修改 |
| 法规错误 | 严重 | 必须修改 |
| 表述不当 | 一般 | 建议修改 |
| 格式问题 | 提示 | 可选修改 |

## 十、使用说明

1. 本清单用于内容发布前的合规性检查
2. 根据内容类型和发布平台选择检查项目
3. 发现问题按定级标准处理
4. 不确定问题及时请示上级
5. 定期更新清单内容
