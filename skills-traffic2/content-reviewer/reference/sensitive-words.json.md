# 敏感词库

## 使用说明

本词库用于内容审核阶段的敏感词检测。分为以下类别：
- **致命级**：必须删除或修改，否则内容不能发布
- **严重级**：需要修改，影响内容质量
- **一般级**：建议修改，可优化表达
